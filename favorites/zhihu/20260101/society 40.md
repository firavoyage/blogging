小润出省，大润出国。

国内是个自上而下的结构。很多地方，资源本身多少，该收到多少补助，该捐出去多少，该有几所大学，体制内拿多少，体制外怎么办，都是上头说了算的。对普通人的影响就是，你一出生就被安排的明明白白，某些地方作为普通人，成长过程中你会觉得不对劲，感觉很窒息（比如山河四省）。

比如我土生土长的浙江，小时候觉得我们这么有钱又重视教育，怎么只有浙大这一所？长大后才发觉：浙江能有几所重点高校，北京说了算，浙江自己说了不算； 一个经济（相对）发达地区，面上只会给你一所，别闹，得“顾全大局”。

最好是原本base在北上广深江浙沪+省会的人，去发达国家赚美元，记得保留随时回国的权利，通过货币消费差和国内高效率（尤其是医疗），你会得到一个更快乐的人生。

出不了国的也没关系，研究下各省福利，弄个双省套餐—-我给我某朋友的建议是北方一个家，南方一个，合理利用两地长处，过愉快的双城生活。

刚看完影视飓风这个视频，真的是狠狠感触了。

说实话，我做过类似的工作，和论文代写类似，是代码写手。




大部分都是C/C++/Python/Matlab/或者是工业软件这些，他们要什么，我就去学什么，主要是欧美和新加坡高校的学生。

大家作为旁观者，可能更多是审视这个行业的对错。但作为一个切切实实干了三年的人，我想从**“底层写手”**的视角聊聊这个行业。




那是大一的时候，我意外通过一些方式接触到了这些中介，他们的单子主要来自新国立、南洋理工、帝国理工这些学校，还有其他欧洲国家和中东等等。说实话，做他们的作业真的是一种折磨，但是也是一种享受。




折磨在于难度普遍来说都特别大，从操作系统内核到前沿深度学习，从简单的数据分析到复杂的多模态这些，随便拎出来一个大作业，工作量和深度都堪比国内的本科毕设。




享受在于，你会惊叹于他们作业设计的合理性和前沿性。说实话，作为一个大一的学生，第一次看到ViT、Slam这些会觉得十分惊叹。而且这些还是强迫你在DDL的前提下，足够优秀地完成这些工作。




从我的视角来看，我没有任何道德负担，我只是有一个任务，要完成xx事情，DDL是什么，然后校对修改校对修改。

![](https://picx.zhimg.com/v2-9fbebe1471c14892a69ba849fc96c639_r.jpg?source=2c26e567)

视频里提到的剥削我也深有体会。

公司会有专门的客服对接，我只负责接单干活。单价由我自己来报，一个平均耗时30h，大概干个三四天。需要反复修改七八次、甚至有一部分需要租卡跑实验的代码项目，我到手通常是300块。至于值不值，你们可以自己衡量。




那时候的我真的以为这个行业很赚钱，毕竟能够年轻的我实现食堂自由。直到后来我和学生聊起才知道，中介对学生的报价是我收入的三四倍。




那种感觉很复杂。




现在回想起来，虽然收入不是很多，但是我通过这份工作偷师了国外顶尖大学的教育资源，我的代码逻辑、科研逻辑，很大程度上是在这些DDL中被逼出来的。




写到这里，其实已经很难再简单地用对或错去评价这段经历了。

**我实实在在地收益，但是又能够真实地看到它存在的问题。**

很多人喜欢把灰色产业的问题，归因到个体的道德滑坡上。但真正身处其中的人都知道，这些产业并不是凭空出现的，它们往往是某些制度性缺陷的自然产物。当评价体系极度结果导向、过程监管形同虚设、而机会本身又高度稀缺时，总会有人被迫、或主动，去寻找捷径。




国内教育的现状也是如此。


我们一边高喊“创新”“科研”“拔尖人才”，一边却用极其粗暴的量化指标去筛选学生：论文、竞赛、项目、排名、奖项。于是**“如何获得这些标签”本身**，逐渐变成了一门技术活，甚至是一门生意。

你会看到有人为了保研，把大创拆成流水线项目或是祖传项目；有人把“互联网+”做成PPT工厂；有人明码标价买专利署名、买论文二作；有人在实验室里从头到尾没碰过实验，却能在答辩台上侃侃而谈。

这些行为在规则之内，却在精神上早已空心化。所有人都心知肚明：这和真正的学习、研究，关系已经不大了。

在这样的环境下，我所做的那份工作，反而呈现出一种诡异的对照。它是灰色的、不道德的，破坏了学术公平，这一点我从不否认。但至少，我写的每一行代码是跑通的，每一个实验结果是真实算出来的，每一次debug都是真实地在和问题本身搏斗。




这并不是在为自己开脱，而是一种更深层的困惑：**当“造假”变得制度化、规模化、甚至被默许时，个体的道德判断还能落在哪里？**




我并不觉得自己比任何人高尚。我只是清楚地知道，在那个阶段，我出售的是脑力、时间和技术，而不是虚假的包装。而这种“实打实的劳动”，在一个高度表演化的教育体系中，反而显得有些格格不入。

当然，这一切并不值得被浪漫化。灰色产业终究是灰色的，它消耗人的精力，也扭曲人的价值判断。当你习惯于在DDL和金钱之间切换，很容易忘记学习本该是为了什么。这也是我后来选择离开的原因。

但我仍然感谢那段经历。 它让我提前见识了顶尖大学如何设计课程、如何逼迫学生思考；也让我更早地意识到，真正重要的不是头衔、项目和奖项，而是你是否真的掌握了“解决问题的能力”。




在这个时代里，保持诚实地思考、诚实地劳动， 也许本身就是一种微弱但真实的抵抗。

**“大不了我就去送外卖/跑滴滴/快递。”**

![](https://pic3.zhimg.com/v2-6aa8ab05c35c5cd6987436db5e0bc072_r.jpg)

铁人三项在过去十年里，是中国职场人挂在嘴边的最后一道心理防线。它是面对裁员、降薪、创业失败时的“社会低保”，是一种只要肯出卖体力与时间，就能换取温饱的安全感。




然而，这种安全感并非理所当然，它仅仅建立在一个特殊的、即将关闭的历史窗口期之上：**中国依然存续的人口红利，叠加尚未完全成熟的自动化技术。**




我断言，未来铁人三项一定会被“消灭”。实际上消灭铁人三项的从来都不是单一的技术，而是技术成本与人矿成本的权衡。

过去十年，我们享受的所谓便捷生活，本质上是建立在海量廉价劳动力和算法极致压榨基础上的“人口红利”。




那么这个窗口期有多久？大胆估计，“纯出卖体力”的托底模式，最多只剩十五年，将彻底成为历史。




当00后、10后全面接管职场，当平均学历普涨至大学水平，当人形机器人不再是科技博主的开箱玩具而是进入量产，大批量进入工厂和街道——**那个我们熟悉的“出卖体力换生存”的时代，将彻底终结**。




站在2025年，我们必须残酷地去推演：**下一个时代，能接住千万普通人命运的“托底行业”，究竟是什么？**




## 一、托底行业的二十年变迁

回顾过去，中国的“社会托底行业”从未静止，它随着国家战略和技术进程在不断迁移。




**2005年**，那是加入WTO后的全速冲刺期，世界工厂的机器日夜轰鸣。那时候的托底行业是普工、小工、农民工。在城市化进程狂飙的背景下，各种电子厂、服装厂、工地像海绵一样吸纳着来自农村的广大劳动力。只要肢体健全，就可以在轰鸣的机器旁找到一个位置。




**2015年**，城镇化进入深水区，互联网开始逐渐渗透到线下。托底行业变成了电话客服、房产中介、理财销售。无数年轻人穿上不合身的西装，依靠极高的社交信用与情绪劳动，在人与人的夹缝中生存。




**2025年，**也就是我们所处的时代，算法下更加原子化，托底行业变成了外卖骑手、网约车司机、快递员。它不需要你懂复杂的社交，也不需要你具备特殊的技能，只需要你服从系统的指令，在这个庞大的物流网络中充当一个可移动的节点。







## 二、为什么铁人三项注定被替代？

很多人认为，只要人还需要吃饭出行，这两个行业就会永远存在。

需求确实存在，**但“人”作为供给端，将不再适配这个行业。**

铁人三项本质上就是对人的摧残。

首先，**人力成本存在剪刀差。**现在看宇树科技、众擎科技的机器人可能觉得还有些笨拙。但是在AI的指数级进化下，通用人形机器人的边际成本将低于一线城市的人力成本。纯体力劳动将不可避免地劣化为**低性价比投入**。




其次，**教育与心理同样存在错配。**前几天我们提起的一个话题就是，为什么大学生不肯脱下长衫？等到2035年，职场主力军是受过高等教育的00后和10后。让他们像父辈一样，长期从事高强度、无成长、被算法实时监工的单纯体力劳动，不仅生理上难以坚持，心理阈值也无法接受。




并且，**收入与劳动强度现在存在强耦合。**现在的外卖模式，收入完全线性依赖于跑单量。这对于追求生活质量的新一代劳动者来说，是一种不可持续的剥削。




那么，未来十年，托底行业究竟是什么？

## 三、蓝领化的白领

下一个时代的“蓄水池”在哪里？别幻想什么高端岗位。既然是“托底”，它就必须具备三个反人性的特征：**需求极其刚性、门槛相对较低、必须由活人来完成。**




未来的就业逻辑将发生根本性大迁移：**从“高强度体力劳动”转向“高强度情绪与复杂交互劳动”。**

简单来说，未来的托底行业，是那些AI不会、资本嫌累、但社会又不得不做的工作。

基于此，未来的就业形态将发生根本性变革：**高密度劳动被大规模替代，人机协作成为主流。**

**我们将迎来一种白领化的蓝领或者蓝领化的白领的工作形态。**




我在这里斗胆预测，未来大概会有三种岗位成为新的社会蓄水池：

- 城市运维体系。

中国已经完成了人类历史上规模最大的造城运动。高铁、地铁、隧道、桥梁、摩天大楼、地下管网，这些庞然大物将长期存在，并进入漫长的老化与维护期。

中国大基建**已经从“增量建造”时代进入了漫长的“存量维护”时代**。




**2. 情绪劳工。**

至少未来十年内，人口结构是未来的铁逻辑。到2035年，老龄人口规模空前，现在的大批量00后都还是独生子女，家庭照护能力会断崖式下跌。这一代独生子女家庭，根本无力承担高强度的照护工作。

例如养老，将从家务事逐渐变为一种工业化的社会服务。

注意，我说的不是现在这种一个月几千块的保姆。

未来的养老产业需要海量的年轻人去提供陪伴、心理疏导、甚至是临终关怀。这极其消耗心力，是典型的高强度情绪劳动。它一定会成为一个巨大的就业蓄水池，无论你愿不愿意。




以及其他泛情绪贩卖行业：广义陪玩、直播（依旧可以）等等。




**3. 人肉标签**

这是最具时代特色的新型托底行业，也是目前正在爆发的前夜。

你们以为ChatGPT是靠自己读书变聪明的？背后是肯尼亚、菲律宾奴工，以及现在越来越多的中国二三线城市里，数以万计的廉价劳动力在没日没夜地给数据打标签。

实际上，中国最底层的职场正在从单纯的体力变现转向基于人的差异化体现。

AI越智能，就越需要人类在极度细分、极度复杂的边缘场景中进行微调。这就是数字时代的富士康流水线。你可能以你想不到的方式，收集大量丰富的数据，但是又不需要你懂编程，只需要有基本的认知能力和理解力。










## 四、最后

当连体力都不再是稀缺资源时，那作为人的潜力真的会被极大地释放——吗？

写到这里，我会感到一丝悲凉：**难道我们努力了这么久，未来依然要靠这些服务性的工作来托底吗？**




托底行业之所以叫“托底”，注定是因为它辛苦、繁琐，是社会分工的金字塔基。这一点，无论在哪个时代都不会改变。

**但我们要看到的是，“托底”的下限正在被技术生生拔高。**

2005年，托底是出卖血汗，我们在流水线上把自己当成**牲口**用；
2025年，托底是出卖时间，我们在算法里把自己当成**电池**用；
而到了2035年，当纯粹的体力苦役交给机器后，哪怕是最基础的托底工作，也开始回归到人的本质。




未来的托底，需要的是你的耐心、你的手艺、你的沟通能力，以及同理心。

这依然会很累，依然是谋生。但在这个新时代里，我们将不再是因为“有力气”而被雇佣，而是因为“我们是人”，拥有机器所没有的温度与复杂性而被需要。

这未必是一个更美好的时代，但至少会更**“像人”**一点。以前我们因为像机器而被雇佣，未来我们因为**“不像机器”**而生存。




这大约就是技术给文明留下的最后一点、带着讽刺意味的人文关怀：**因为它造不出灵魂，所以它不得不留下我们**。

B站犯了一个相当大的错误。

他们用短视频的逻辑来运营长视频。

啥意思呢，我举个例子你就明白了。

假设你特别喜欢一部动漫或电影，想找一些关于这部电影的二创或解析视频来看。

但是这部电影已经有些年头了，尽管它曾经在B站非常火爆，当时涌现出大量优秀的二次创作、剪辑和鬼畜段子等，但现在你可能一个也找不到。你的首页可能永远不会推荐这类内容，而且搜索栏搜索出来的结果也可能一言难尽。

哪怕明明你敢兴趣的这些东西还存在B站的数据库中，但是给你的感觉就好像这部电影从来没有火过一样。推也推送不到，搜也未必搜的到。

这样的例子在B站不胜枚举，B站曾经火过无数的电影，动画，电视剧，音乐什么的，但是只要时间一久他们就完全消失，就好像他们完全不曾存在一样，不留下一点痕迹，你有兴趣可以关注一下你的推荐栏，看看推荐的视频是不是都是最近三个月上传，最多半年，撑死一年，一年以上的视频就几乎不会推给你了。

有人说这不正常吗？热度不是本来就一阵一阵的吗？过时了就不推了不是很正常吗？对于短视频来说确实是这样，但是长视频不是，因为长视频里面很多是有内容的，而优秀的内容是不会随着时间而过气的，是可以维持很长时间的价值的，一个好的长视频即使过个三五年甚至十年往上都会有人愿意看，更不要说一些专门喜欢早期歌曲明星和电影的观众了，B站的问题是这些过去的优秀长视频几乎推送不到喜欢他的观众那里。

长视频的推送逻辑应该是以内容质量为导向，而评定内容质量的标准应该是参考观众的审美和兴趣，而不是推送时间，如果算法觉得我应该喜欢这个视频，那哪怕这个视频是十年前的你也应该推给我，而不是无限拔高时间的权重。

油管正是这样做的，我最近很喜欢一个早年很火的欧美明星，现在基本没什么人关注他，油管天天给我推送他的视频，首页全是他的视频，这些视频质量都很高，但发布时间普遍是好几年前，甚至十年之前的老视频了。如果是在B站，我几乎不可能看到他的推送，我的首页可能完全没有他的视频，因为一年之内几乎没有有关他的新视频上传。

长视频最大的问题在于优秀的长视频的稀缺性，一个好的影视解读或者考古可能几年才出现一次，但油管会将其保存起来推送给每一个可能喜欢看的观众，B站最多推送一年就不推了。

举例就是李子柒，在她停更的这三年全网粉丝都在掉，但是油管反而涨了三百万粉丝，因为在她停更之后她的所有视频依然在被油管不断推送给所有可能喜欢看的观众。

在B站一个视频绝大部分的播放量都在上传的前24小时最多前一周，但在油管，即使上传几年的视频也依然可能有几百万的增长。

体现出来就是B站越来越没意思了，因为你觉得有意思的视频可能是几年前的，B站的算法根本不会推给你。

有内容的长视频在用户体验上变得越来越稀缺。这样直接导致长视频更加打不过短视频，最后只能被迫向短视频转型。

这点B站和知乎很像，二者都是内容平台，但知乎是只要觉得你可能感兴趣的甭管多久之前的都推给你，我甚至收到过十年前的知乎文章。

B站是只推最近上传的，甭管之前有质量多高的视频。

想象一下，如果知乎也只给你推最近的东西会怎样，结果必然是内容的极度欠缺，原本的内容社区优势逐渐失去。




这样一想就能看出B站目前这个推荐机制有多荒谬。

一个大学生，从小是标准好学生。小学中学被要求写学雷锋的事迹、写我的理想职业是当人民教师、警察、医生、写感恩父母感恩祖国。到了大学，学的是新闻、中文、马政相关专业，课堂上学的是传播理论、典型事迹、舆论引导、文学修辞。毕业后，直接通过考试进入宣传部门或机关单位。一生中和现实社会真正的接触，多半停留在校园和短暂实习之内。被拖欠工资、负债缠身、创业失败，这些在普通人生活里关联度极高的东西，对他们来说往往只是教材里一笔带过的所谓社会普遍问题，而不是浸入骨头的亲身经验。

然而一进单位，他马上被要求去写接地气的报道、百姓故事和基层典型。领导希望看到的是真实感人、充满温度、充满正能量的文字，这时候，这位年轻写手面对的，是一种非常尴尬的处境：他既没有足够的生活经验积累，也没有自由讲真话的空间，只能在有限的语言工具箱里，给每一个遇到的人、每一个看到的故事套上现成的结构。一个人很穷，那就写家境贫寒却自强不息。工作环境很差，那就写条件艰苦但默默奉献。面对巨大压力仍坚持，那就写不向命运低头。最后可能再来一句在组织的关怀和社会各界的帮助下，他脸上露出了久违的笑容。至于那张所谓笑脸背后，是否还裹着焦虑、屈辱、不甘、愤怒，这些东西在稿子里是完全没有位置的。因为一旦这些东西写进去，就会变成难以被升华、甚至可能指向制度追问的危险内容。

对刚进系统的年轻写手来说，他确实不太有能力把复杂的结构性苦难写清楚，在他有限的生活经验里，最大的想象力也只能是把别人承受的一切翻译成值得敬佩的精神。这种叙事线路既安全、又熟悉。他又在各种培训中被告知要坚决防止负面和避免渲染灰暗情绪，自然会本能地把不能见阳光，永远在阴暗面的东西抹平。慢慢的，歌颂苦难就从一种写法，变成了一种默认的选项，任何让人不舒服、甚至想问为什么会这样的现实，都可以装进平凡英雄的模子里，从制度必须回答的问题，变成用来教育别人要坚强的鸡汤故事。

久而久之，就出现了一幅极具反讽意味的图景：越是生活在苦难和压力里的群体，越是在宣传里被写得单纯、乐观、感恩。越是现实中充满无奈和愤怒的人，越频繁地被塑造成不抱怨、讲奉献、埋头苦干的典型。甚至在很多公文体的叙事报道里，所谓的苦不再是一种需要减少、需要纠偏的现实，而是一种可以用来证明高尚、证明意志力、证明忠诚度的素材。

越苦，越适合被歌颂。越能忍，越值得被端上台面作为榜样。越能在糟糕条件下坚持工作，越说明他的道德高尚。

于是就有了人人熟悉的固定桥段：某人在海拔几千米的偏远地区坚守几十年，最终终身未婚，把青春全部献给那片土地。某个乡村医生翻山越岭给村民送药，雨雪无阻，从不谈报酬。某个老师在破旧的教室里教书，一边忍受疾病，一边咬牙坚持在岗。某个工人带病加班，直到倒在工作一线，用生命诠释了责任。这些故事并不是虚构，很多人的确值得尊敬。但在这种叙事里，很少有人去问另一层问题：为什么会有那么多岗位常年处在缺人、缺钱、缺保障的状态？为什么基础设施、公共服务、劳动保护在某些地方迟迟难以到位？

这其实就是歌颂苦难的核心问题：它并不只是多写了几句煽情的话，而是通过不断把苦转译为德性，把原本属于制度责任的一些问题，重新包装成个体高尚的证明。苦难不再是一个必须被减少的现实，而变成了一个可以反复被调用的资源。只要有人愿意扛着不吭声，这个系统就可以继续不必回答那些尴尬的问题。

但这种行为本质上就是掩耳盗铃，是把自己的脑袋埋在土里，因为不管你怎么解释，在这个教育、技术越来越大发展的时代，越来越多的人会觉醒：很多所谓的苦并不是命运的考验，而是极具体的制度结果。医疗和医保的规则，区域发展差距的长期固化，劳资关系里的力量失衡，社保体系的漏洞，地方财政压力向基层摊派的方式，教育资源如何通过房价、学区一步步筛人，这些都不是抽象词，而是每个月账单、每一年人生选择的背景板。

但当他们周围的苦难被一遍遍讲成高尚的牺牲和感人的奉献时，普通老百姓感到的不是被看见，而是被利用——被要求继续为这一整套结构背书。

你问一个在流水线夜班熬到脱发的人，他最大的愿望是什么，他可能说的是别再突然降薪、别再拖欠工资和别再动不动裁员。宣传稿里呈现出来的东西，却可能是他主动申请夜班，只为多一点收入给孩子能够提供更好教育，对未来充满希望。

前者是人的真实欲望和恐惧，后者是话语系统需要的正面形象。谁都能看出这两者之间的落差，于是观众读到这样的故事时，直觉上感到一种冒犯：你不是在诚实地描述我的处境，而是在利用我的处境，完成你自己的政治叙事。

这也是为什么，正能量故事这几个字，在很多人心里慢慢变了味。

问题不在于正面本身，而在于那种用温情的包装封住裂缝的写法：现实里的痛点被小心地绕开，只留下可以安全流泪的部分。人们厌恶的，是这种对现实的选择性凝视。甚至当理想与现实的割裂感越来越大时，个体就会往往会产生强烈的挫败感和无力感，从而引发痛苦的情感体验。

最糟糕的结果就是老百姓对于政府宣传的态度大幅度转变，最终演变为真正相信这些故事的，只有那些写稿的人和审稿的人，他们真心觉得这样能鼓舞人心。

而那些被写的人、围观的人，看多了之后也会越来越自己清楚：你们写的故事根本不是我们，而是一种被你们想象出来的所谓理想人民。

遥不可及。

孔乙己的长衫这件事情为什么会被全体国民群嘲，本质上也是这个逻辑被戳破。那篇文章的致命之处不在于技术上的文笔问题，而在于它试图把当代年轻人的经济困境、身份焦虑、结构性机会收缩，粗暴地翻译成一个道德问题：你找不到工作，是因为你还穿着精神上的长衫，不愿放下读书人的架子。你不肯去端盘子，是因为你眼高手低，不肯吃苦。现实层面的产业升级、岗位减少、就业门槛抬高、地域发展不均，在这套叙事中统统消失，只剩下一个被道德化的指责：是你不对。

这种套路，在传统宣传中非常常见：把结构问题转译成个体性格问题，把制度层面的责任转嫁到个人选择上。最终的确结构的阴影消失了，每一块灰暗都被找来铺成高尚精神的背景。

但这种简单粗暴最终往往会弄巧成拙，为什么？

因为互联网时代有一个前所未有的变量：每一个被代表的人，都能开口说话。以前你在电视里看到一个感人事迹，你很难知道他本人怎么想。现在，这个人的朋友圈、他同事的吐槽、当地网友的补充信息，很快就能把这个故事拆开。你把结构问题删掉，观众会在评论区补回来。你把个体的愤怒掐掉，观众会用段子和嘲讽把它重新呈现出来。宣传系统面对的，不再是一个被动接受的观众群体，而是一群会反向审核你的话语、会集体给你打真实性分数的人。

也就是说，过去那种由上而下的单向灌输模式，已经在技术层面失效了。

问题是，很多宣传工作仍然用旧的心态在操作：只要版面够大、话够多、情绪够煽，就能压过嘲讽、覆盖住质疑。可现在的信息结构是碎片化、去中心化的。你说你的，我吐槽我的，你删你的评论，老百姓就会在另一个平台、一个私密群、一个二创视频里接着笑你。

宣传机关发布的每一条正能量故事，其实都在同时接受一个暗场的评审：

这次你有没有真正理解我们？有没有把自己的责任从故事里抽走？有没有在借我们的苦证明你的正确？群众的嘲讽，是在用最原始的方式表决：这套说法，我们不认。

所以，今天的宣传乱象，有一部分是认知延迟：现实已经进入了人人有麦克风时代，话语权已经分散，而宣传体系内部还是仍然认为，掌控几个大号、几块版面就能定调。还有更深一层，是立场错位：宣传习惯站在一个教育者的位置，认为群众需要的永远是被教导、被激励、被安抚，而不是被诚实相告事情确实很难、我们也有压力、有些问题一时解决不了。这种立场，导致宣传文字有一种天然的俯视感：

你们的困惑，是因为没看清大局；你们的焦虑，是因为不理解现实；你们的抱怨，是因为心态不阳光。

但今天的普通人，不再把自己当成被教育的对象。他们接受过的教育水平整体抬升，能接触到的信息极大丰富，对现实的细节、世界的运行方式、制度的局限性都有自己的直觉判断。当宣传还在一贯用家长式口吻说话时，他听到的不是关心，而是一种默认他不懂事的姿态。这种姿态一旦被识别出来，嘲讽就几乎是本能反应：你既不愿意把真话说全，又要站在高处教育我，那我干脆拿你当反面教材来玩。

在这种情况，我们也的确看见了一些部门开始反思：但是嘴巴上虽然说着要变得更接地气、要下基层要讲群众语言。单位组织年轻记者去社区、乡镇、工厂体验生活。可有时候所谓接地气，只是加点方言、拍几张现场照、写几句生活细节。

就像这次所谓的包的和志在必得事件。虽然改进了，但还是被人不断被人群嘲。因为叙事框架不变，那产出的只能是形式上的接地气。

本质依旧是空中俯视。

真正的接地气，首先意味着把人当人，而不是当素材。所谓群众故事，不应该从我能不能把他写成先进典型出发，而应该从他的生活到底是什么样，他究竟在怕什么、担心什么、愤怒什么、期待什么出发。真实的普通人未必高尚，也未必时时乐观。很多时候他们是矛盾的、自私的、犹豫的、在多重压力里反复妥协的。如果宣传只允许呈现被净化之后的那一面，就等于宣布：真正的普通人是不配进入镜头的，只有被加工成理想类型的人，才有资格被写进稿子。

其次，真正的接地气离不开对问题本身的承认。一个社会要形成稳定的基本信任，不是靠要求大家只说好话，而是靠在重大矛盾和真实焦虑面前，至少能被看见、被命名。宣传如果永远停留在困难只是暂时的和问题只是个别人身上的，甚至干脆绕开关键矛盾不谈，只会让官方话语和民间感受之间的缝越来越大。很多人失去信任，并不是因为它在某件具体事实上的错误，而是因为他们反复感受到：

你明明知道事情远比你说的复杂，却总装出一副一切尽在掌握的样子。你明明可以承认某些环节确实做得不够好，却永远只强调已经取得了巨大成就。这中间的割裂感一旦积累，默认态度就会从半信半疑自然而然滑向本能不信，哪怕偶尔说了真话，也很难再被当作真话。

再往深里看，就和刚才说的一样，信息化时代之下，今天的宣传乱象还有一个最关键的背景：媒体环境和信息结构已经发生了根本变化，但很多宣传部门的基本工作逻辑，仍然停留在我说你听的单向灌输时代，习惯仍然以为：只要把声音放大、把反对声音压下去，事情就算解决。长此以往，宣传系统容易陷入一种自我麻醉：内部汇报时看到的是阅读量、点赞数和正面评论数，数据光鲜亮丽。但在另一个场域里，越来越多的人是抱着看笑话的心态消费这些内容。表面数字好看，底层信任却在不断流失，这是一种更隐蔽、更危险的失效。

再把视角拉回那个刚毕业的大学生身上。他也许真心想写点不一样的东西，想把基层看到的矛盾、尴尬、不公平写出来。但他手里握着的是一套只有安全版本的话语工具，他面对的是一个把风险可控放在首位的考核体系。每一个句子写下去之前，他要先经过一轮自我审查，再想到上级的审读，再想到可能出现的舆论风险。时间久了，他自然会形成一种本能：先学会把锋利的东西磨钝，再学会主动去寻找可以升华的角度。当一个写作者的本能不再是把自己看见的真实全部写出来，而是在现实里搜刮可以用的正能量素材，他本人就难免变成那套叙事机制的一部分。

这就是为什么，简单的所谓下基层和改文风远远不够，治标不治本的行为往往最后都会沦为作秀。

可以说，真正要扭转今天宣传被群嘲、被疏离的状态，至少我们需要有几层底线。不能再把苦难当成可以随意消费的素材，而要重新把它视作需要认真对待、努力减少的现实困境。不能再把结构问题一味推到个体德性上，而要正视制度设计、资源分配、权力运作中确实存在需要持续修正的地方。不能再习惯于站在教育群众的姿态上高高在上地说话，而要不断承认群众本身的判断力，把他们当作有能力理解复杂现实的公民，而不是只能被灌输简单故事的对象。

很多人对宣传的期待其实很朴素：不是要你每天报忧，而是希望当现实确实存在难以回避的痛点时，官方话语不要那么无知、那么冷漠、那么急于自我辩护。如果一个体系连承认问题的勇气都没有，那么它此后每一篇正能量报道，在很多人眼里都像是在给问题抹粉。群众嘲讽的，从来不只是某一篇写得拙劣的稿子，而是那种我们什么都懂，但什么都不会说破的高姿态。

在我的理解中，宣传这两个字，按最朴素的理解，本质上就是把事情说明白，把道理讲清楚，把方向说出来。可现在很多人一听到这个词，脑子里冒出来的却是虚假、套路和官样文章。这本身就是一个需要被当成严重问题面对的信号。要让宣传重新有力量，没有什么神奇捷径，我们只能回到最基本的一点：敢于面对真实的生活，哪怕里面充满不完美、矛盾和暂时解决不了的难题。一个不敢说真话、不敢承认问题、不敢展示复杂性的宣传体系，只能越写越空，越写越被当成笑料。相反，如果它哪怕只退半步，从教育别人变成和别人一起看现实，愿意在必要的时候坦诚地说一句这里确实有问题、这件事不好解决、我们也在摸索，愿意让那些真实的困惑、愤怒、犹豫也在镜头里出现，那么它也许才能慢慢赢回一点信任。

群众并不是天生爱嘲讽宣传，他们只是已经厌倦了被当成听众，只会被要求鼓掌，却得不到真正的对话和回应。当宣传只想要掌声时，它收获的多半是沉默。当它习惯于高高在上时，它等来的往往是冷笑和段子。

要扭转这一切，不是在文章里多加几个网络流行语，不是在起标题的时候搞谐音梗，不是在视频里配几段轻快的配乐。而是从根本上换一种说话姿态：

少一点替你总结人生经验，多一点听你把话说完。少一点对苦难的浪漫化，多一点对问题本身的直视。少一点对群众的想象，多一点和他们处在同一片现实地面的站位。只有这样，宣传才有可能真正接上地气，而不是继续在话语的高空里自转。

整个社会都想给你洗脑“你的人生需要被安排”

东大社会是一个自上而下掠夺的模式，软硬兼施的目的只为了让你不断参与生产，不断提高生产的效率和可持续性

软的就靠亲属关系构成的【意识形态认同圈】，靠社会媒体舆论输出吃苦是福的观念，而大多数人的父母亲属的价值观世界观人生观又几乎都来自媒体

所以你没有安稳工作就给你上压力，给你爸妈上压力，说你不孝顺，说你不务正业，说你没有走在【正路】上，其实本质是异化劳动的“正路”

又因为年轻人从小所接受到的教育就是把自身的价值和存在意义依赖在他人的身上，所以他们在受到外界质疑和评判时会感到急剧焦虑和痛苦，这种焦虑与痛苦不断驱使着年轻人逃避自己的【主体性课题】从而不断在六亲和社会舆论下就范

而硬的方面，就是制度性的结构性的控制

它不像软规训那样通过情感与舆论，而是通过【时间】、【空间】与【生存条件】直接对个体进行规训

首先是时间的控制

学校从早到晚的作息、工厂与公司的考勤制度、城市的高压节奏，

共同构成了一种对时间的殖民

人一旦被时间的外部节奏所统治，就失去了对主体性和生命力的感知能力

于是，“早八晚十”成了勤奋的象征，而不是被剥夺的迹象

人们开始主动为自己的疲惫感到骄傲，这正是硬控制成功的标志

其次是空间的控制

住房、户籍、通勤距离这些不是单纯的生活条件，而是社会对你行动范围的封闭性规划。

人被迫在租金、地段、学区的计算中失去迁徙自由，最终被固定在某个社会层级的职位格子中

在英文里，职位用“occupation”一词来表示，而该词的动词形式“occupy”的原意就是指主体将自身固定在某一位置不动的过程

再者，是生存资源的控制。

当医疗、教育、养老都与“工作挂钩”的时候，“参与生产”就不再是选择，而是生存的前提

这种结构性的绑定就让你不可能真正“停下来”，一旦脱离系统，你就不仅失去收入，还失去被社会承认的合法身份

这就是为什么所谓“gap year”在东大社会被视为一种禁忌

因为对于以效率和可持续性至上的社会机器而言，不能转动的齿轮只有被扔出机器取而代之的命运

一个社会走向反智横行，表面上看，是大家越来越不爱读书，不相信专家，动不动骂粉红、公知、五毛，社交媒体上流行的是情绪和段子，而不是分析和论证。

可要真把问题追问到底：

究竟怎样的社会，才会一步步把反智当成理性选择、变成时代气质？答案从来不是人变笨了，而是制度结构、经济处境、文化情绪和政治动员慢慢叠加，把认真思考变成一件成本极高、收益极低的事，把简单粗暴变成最划算的生存方式。

如果说得抽象一点：反智不是知识和无知的对立，而是事实与身份、真理与权力之争的某种表现。当一个社会越来越相信真相不重要，站队才重要，反智就从边缘情绪，升级成主流叙事。

最直观的一个层面，是人们普遍而持续的不安全感。一个人只有在基本生活可预期、未来大致看得见的时候，才愿意给复杂思考留出空间。你可以花时间和有耐心去了解现代税制、产业结构、国际局势，也愿意承认现实十分棘手、问题异常难解，因为这种承认不会马上威胁到饭碗和房租。

但如果一个社会里，大量人的处境是：工作随时可能被优化，房贷压得人喘不过气，父母养老、子女教育都没着落，一个意外就能摧毁全家积蓄。

那么，面对现实的复杂就不再是一种理性的清醒，而是一种心理折磨。你越是了解结构问题、财政困境、产业空心化，就越清楚这不是靠个人努力能解决的。这种透彻反而会转化成无力感，甚至绝望。

在这种日常压迫之下，人会本能地偏爱简单解释。

简单解释有两个好处：

第一，它把复杂结构压缩成几个坏人、几个阴谋，能迅速提供情绪出口。第二，它不要求你改变自己，只要把愤怒指向一个外部靶子。

于是，都是资本阴谋、都是某国操纵、都是专家勾结的语言，会比任何需要耐心和证据的分析更有市场。

反智在这里，并不是认知能力下降，而是一种自我保护机制。与其真的承认世界极其复杂，而我几乎无能为力，不如相信真相其实简单，只是被遮蔽，这样至少保留了一点心理上的优越感：别人是被蒙蔽的，而我是看穿一切的人。这种通过否定知识来挽救自尊的机制，会在不安全感持续放大的复杂社会里，重复出现。

另外，反智主义盛行的土壤，还有重要的一点就是社会流动通道的堵塞和向上叙事的破产。许多人之所以愿意接受漫长教育、忍受考试机器，是因为相信知识有用。哪怕这种有用极其功利，只是意味着更高的收入、更稳定的工作，它也在某种程度上支撑着对知识和学校的尊重。

但当现实一次又一次证明：高学历并不自动对应体面生活，基础岗位薪资停滞，专业与岗位严重错配，很多人读到研究生、博士也只能卷进低薪竞争，教育曾经承诺的回报兑现不了，整个社会对知识的态度就会悄悄变形。

教育体系的变形，则在更深处加固了这种趋势。很多地方的教育长期以应试为中心，把知识简化成标准答案。学生从小接受的教育训练，是如何在有限时间内推断出命题人的意图，而不是如何提出自己的问题。课堂之内，老师有标准答案，课堂之外，参考书有标准套路，一切都围绕答对题展开。这样的教育即便能培养出无数高分考生，却很难培养出真正具有独立判断能力的所谓现代意识公民。

因为它从一开始就不鼓励怀疑和追问，也不训练如何在信息纷繁时辨别何谓可靠与不可靠，只训练你如何在规则既定的情况下优化自己的表现。一旦离开了考场，那些规则变得模糊，信息来源五花八门，许多人就完全失去了抓手。

更糟的是，当这样的教育又被彻底推向功利化，知识进一步被压扁成就业工具。选择专业时，人们首先问的不是我对什么有兴趣和社会长期需要什么，而是哪个最赚钱或者哪个最稳妥。一切课程的存在，都必须用市场上的薪水来证明合理性。一开始，是读什么专业更能赚钱，所以人文学科、基础科学慢慢被视为没用，最终被边缘化，这其实不只是学校内部的事情，它会让整个社会失去一套训练自我反思和理解他人的重要工具。再往后是读那么多书也就那样，所有读书人的象征资本被不断打折消费。再往下，就变成了读书不如早早出去混社会，读书本身的价值遭到怀疑。

知识从可能的阶梯，变成无效的赌注。

在这种环境里，现代的知识被不断削去内在价值，只剩下有没有用。一旦现实中的回报不如预期，读书无用论就有了强大的群众基础。而一旦读书无用成为普遍经验，人们自然而然会延伸出思考无用和复杂无用的最终结论。功利主义教育的最终产物，本质上不是理性的社会，而是一代人对理性本身这个概念心生怀疑。甚至于当一代人觉得自己被这样的赌注戏弄之后，对知识的怨气就会转移到对知识人的怨恨：你们这些人当初鼓吹读书能够改变命运，现在却端着铁饭碗教人卷书本。

这个时候，反智带着明显的阶层情绪。它不只是我不信你说的东西，而是我不信你这个人，我不信你这一类人。教师、学者、研究员、医生、专业记者，不再被看作掌握知识的群体，而被最终归入吃体制红利的既得利益者。他们说出任何复杂观点，都会被先扣上屁股不正的帽子，再看内容。随意攻击他们，不是因为不同意观点，而是因为攻击他们本身就能提供一种象征性的阶级复仇。

这种对知识群体的怨恨，与其说是一种价值判断，不如说是被堵死的社会流动在精神层面的爆炸。当上升通道狭窄到只能容纳极少数人，而大多数人就算努力都掉进内卷的黑洞，读书改变命运的叙事就变成讽刺。人们不再把知识看作共同资源，而把它看作一个小圈子的特权，随之而来的，就是反智式的翻盘：既然我进不了那个圈子，我就否认那个圈子的正当性。

另外，在这种情况下，最容易出现的一个状况，是权威机构的信用崩塌。一个社会可以接受普通人不懂太多，但不可能接受专业机构也靠不住。传统上，大学、研究机构、媒体、法院、专业协会，是公共理性的重要支柱，它们的功能不是绝对正确，而是为社会提供一套相对可靠公正的事实框架和判断标准。可如果这些机构本身，日益常态化被曝出造假、腐败、利益输送，或者在重大公共事件中不断前后矛盾、遮遮掩掩，那么公众对它们的信任必然急剧下跌。人们的经历往往是这样的：某次事故，官方说法翻来覆去变更。某项政策，表面用的是科学、数据语言，实质却是利益妥协。某个丑闻，调查却是久拖不决、处罚极其轻微。某个灾难，却被丧事喜办。每一次这样的事件，都会向社会发出一个信号：所谓专业与科学，不过是权力语言的外衣。

长此以往，怀疑专家就从一种健康的批判态度，演变成对一切专业话语的本能拒绝。气候学家讲气候变化，有人马上说：是不是拿了环保产业的钱。流行病学家讲防疫，有人马上说：是不是受制于某些利益集团。经济学家讲财政约束，有人马上说：是不是替资本说话。关键点在于，这种质疑不要求具体证据，它直接从权威可能被腐蚀推演到所有专业话语都是阴谋。一旦这种逻辑在社会中广泛传播，反智就不再是迷信民间智慧，而变成一种带着正义感的审判：打击权威的最廉价武器。

当今社会下的信息爆炸但真正的公共讨论空间却在萎缩又助长了反智主义的盛行。表面上，互联网是知识的盛宴，人人都可以搜索资料、查看原文和专家的说法不再垄断。但许多现实却是恰恰相反：当信息总量大到人类注意力已经无法承受的时候，信息本身就不再会按照是否真实、是否有逻辑被筛选出来，而是不断按照是否抓人眼球、是否能刺激情绪被放大。

平台算法会自然偏好那些简单、极端、激烈的观点，因为它们更容易引起人们的转发和争吵。温和、复杂、一步步铺陈论证的专业表达，则往往被淹没在洪流中，甚至连送到你面前的机会都没有。

在这种机制之下，好好说话的人会感觉越来越吃亏。他可能静下心花几天整理资料、核对数据、写出一篇对某件公共事件的冷静分析，结果阅读量不及一个几百字的咆哮帖的零头。久而久之，要么退出，要么为了流量也学会喊口号。这时候，公共言说的生态变成谁更会煽动，而不是谁更能讲明白。从外部看，这就是反智声音越来越大。从内部来看，则是理性声音被算法和注意力经济主动边缘化。

这种文化风气，背后还有一个更深的心理机制：怨恨。尼采说过，被压抑而无处安放的怨恨，会倾向于通过精神上的翻转价值来寻求补偿。现实中，很多人在知识、收入、地位上都感到自己处于劣势，却又找不到现实途径改变，于是只能在精神上进行所谓的价值翻盘：既然我不可能比你更懂，就干脆否认懂这件事本身的价值。既然我无法攀登你所在的象牙塔，就说那座塔从根开始就是腐烂的。于是你们虽然有知识，但你们不懂真实生活、我虽然没读多少书，但我比你们更接地气、更有良心，就变成在反智话语中反复出现的句式。

最后，还有一个不怎么能说的重要因素。

反智并不总是自发长出来的，它经常被有意激活、放大和利用。对古今中外任何一个肉食集团来说，反智都是极有用的资源。因为理性讨论总是意味着必须解释政策、公开数据、接受批评、承认不确定性，最终达成共识，但这些都非常麻烦。而反智式宣传只需要几个简单词：

敌人、阴谋、背叛。

人们在这种情况下，如果一旦习惯于反智逻辑，社会气氛会发生微妙但深刻的转变。复杂分析不再被视为必要，而被视为软弱、动摇。任何提醒风险的人，都很容易被骂成唱衰、泄气和给敌人递刀子。公共讨论被压缩成立场秀：用词越狠，越显得坚定。表达越简洁，就会越显得接地气。

这种氛围下，严肃知识很难被容纳，因为它天然拒绝简单化。而且这种氛围还会相互影响。如果上层始终沉迷于简单敌人叙事，对真实世界的复杂和长期成本缺乏耐心，只需要的是立即可见的政策效果和话语胜利，而不是缓慢繁琐的制度修补。这种自上而下与自下而上的反智彼此强化，最后构成一个闭环。现在的美国和欧洲其实就是这个样子的典型。

把这些条件放在一起看，一个反智横行的社会，大致呈现出这样的结构性景象：经济社会高度不稳定，大量普通人觉得自己的努力无效。教育和职业上升通道狭窄，知识回报缩水。传统权威机构失信，专业话语被一视同仁地视为权力工具。信息环境由算法和商业逻辑主导，短平快的情绪内容席卷全体注意力。教育系统长期只训练应试技能，忽视思考能力和自我反省。文化心理中积累着对精英的怨恨和对差距的羞耻。政治和舆论场乐于利用敌友叙事，以简单粗暴压制复杂讨论。

在这样的综合环境里，你要在公共空间里坚持一种认真的、审慎的、愿意承认不确定性的姿态，几乎等于拿自己的生活和心理健康去冒风险。

你不仅要面对来自算法的冷淡、来自舆论的误解，还要承担来自各方的敌意：有人骂你站在精英一边，有人骂你愤世嫉俗，有人骂你专挑毛病，连你自己也经常会怀疑：费这么大劲解释问题真的有用吗？

当用脑子成为一种需要巨大心理抵抗力的行为，而跟着情绪走却能立刻获得归属感和存在感，反智就成了最理性的不理性。在这种意义上，反智不是笨，而是现实激励结构扭曲后的正常反应。

所以，如果说这样的社会越来越反智，实质是在说：

这个社会的制度安排与文化逻辑，正在系统性惩罚那些愿意费力思考的人，系统性奖励那些顺着情绪起哄的人。

久而久之，即便有人仍然重视理性，也会出于自我保护而退到私人空间，只在小圈子里讨论，不再试图影响更大的公共舆论。公共空间由此被情绪主宰，集体决策就越来越容易被误导。

这就是反智主义最大的问题所在，这种状态并不会只停留在口水战层面。

因为反智习惯一旦深入社会肌理，就会直接影响一个共同体面向风险和危机时的应对能力。当一个国家在面对经济衰退、技术变革等大规模挑战，一个社会如果习惯用简单叙事取代专业研判，习惯用立场指责取代证据比对，就很难形成有效的集体行动。要么在犹豫和争吵中错失时机，要么在情绪裹挟中做出短视决定，然后在下一轮冲击中付出更大代价。

要逆转这种趋势，靠喊口号的多读书显然是无济于事的。但这也并不意味着反智是不可逆的命运。只有理解了它的结构成因，才有可能在具体层面做一些逆向工作。比如，在经济与社会政策上，真正减轻普通人的长期不安全感，提供稳定而透明的基本保障，会给理性讨论留出空间。在教育上，减弱应试至上，把更多心力放在批判性思维、逻辑训练和信息素养的培养，而不是只教孩子如何答对题。在媒体与平台治理上，通过制度设计让那些有证据、有论证的内容，得到相对更多的呈现机会，而不是完全交给算法追逐流量。

另外，同样重要的是，知识生产者本身也需要反思。反智横行并不意味着知识界完全无辜。许多时候，专家确实习惯于在封闭圈子里说话，忽视表达方式、忽视与普通人的经验和语言建立连接。有时也确实容易陷入某种精英傲慢，把一切质疑都简单归类为所谓的愚昧，而不思考反思自己是否解释得不够清楚、是否在利益关系上不够透明。要在反智氛围中守住一点理性，要求的不只是公众多读书，也要求知识阶层学会更开放、更谦卑、更愿意在复杂现实中与人同行，而不是站在道德高地上俯视一切。

要改变这一点，就必须一点点调整这些选择的代价：让说真话的成本没那么高，让认真分析的人不至于被完全孤立，让有修养的表达不至于淹没在污言秽语里。

当一个社会能够做到：在面对重大公共议题时，大多数人哪怕意见不同，也愿意先听对方把话讲完，再去争论立场。在学校里，孩子们不仅被教会背标准答案，也被鼓励提出不同观点并为之寻找证据。在大部分媒体和网络上，那些真正有洞见、有深度的文字和声音，不必装疯卖傻，也能获得他们基本的关注。在政治上，表达复杂性、承认不确定性、不轻易作出绝对承诺，不至于立即被淘汰出局。那时，反智才会慢慢退回它应有的角落，变成个别人的选择，而不是整个时代的气质。

反智从来都不只是有人有没有读过书，学历到底高不高那么简单，而是牵涉到谁能决定叙事、谁能定义现实、谁能在公共舞台上被当回事。一个越来越反智的社会，其实是在说：我们不再相信真理是通过争论和论证慢慢逼近的，而只相信力量和情绪能立刻决定自己的一切。当越来越多人用这种方式理解世界时，再聪明的个人，也很难独自保持清醒。因此，与其抱怨人怎么越来越笨，不如认真追问：

我们正在打造怎样的制度和文化，让思考变得如此不受欢迎？只有回答了这个问题，反智才不至于成为共同命运。反智才有机会慢慢退潮，而不是继续被当作一种愈演愈烈的时代标志。

因为罗翔是中国比较少有的自由派律师

简单来说就是他的核心观点是限制政府和司法机关权力，也就是限制公权力，这在中国人之中是很少有的，哪怕在律师之中也是少有的，所以很多人一碰到跟自己观点有冲突的地方就跟炸了一样，因为跟自己从小学到的不一样

很多人说我也同意限制公权力呀，不，其实你并不同意

限制公权力，落到一些不那么敏感的地方还好，比如说我不能随便就进你家门扇你两巴掌，但如果落到一些敏感的地方就不一样了

限制公权力，意味着提高入罪门槛，意味着废死派，意味着取消前科制度

这三者可以说是罗翔的观点里面最具争议的地方了，但其实这跟他的核心观点是完全一致的，那就是限制公权力

大多数人看似赞同限制公权力，但骨子里其实压根不赞同，自己还以为自己赞同。简单点的，你肯定赞成警察没有搜查证不能随便进人家里搜查吧，但你大概率只是口头赞同，不信我举个实例，一巴士司机载着一车人到桥上，车辆失控掉河里了，人全死了，车上也没有录音录像设备，现在公安想调查动机，叫司机妻子到公安局配合调查，妻子可以拒绝吗？没搜查令到司机家里想进门调查，妻子可以拒绝警察进门吗？在没有任何证据的情况下你赞成警察办搜查令搜查司机家里吗？

所以中国人绝大多数还是希望有个大政府，把所有事都管的事无巨细，出了事情瞬间就解决，而自己是守法公民，大棒是肯定不会落到自己头上的

他从小学时便主张压岁钱自己存，他妈妈一直不同意，最终，从初中开始自己管理压岁钱，用我多余的电话卡开通了支付宝，收到现金就会让我给他换成电子货币。他每年压岁钱2000多点，合计有11000左右，高一高二获得两次奖学金，合计6000，因为考上了重点中学的重点班，我答应每月给100零花（有时我们都忘了就没给。餐费交通费另计），估计他能支配的金额20000左右。

这里我则想提一下这个问题的国际情况，因为不止国内，在国际上也有很多人观察到工程和科学类专业有大量的二次元（和福瑞）。

近十年前MyAnimeList就有一个关于学生用户的专业调查[1]，结果是参与者中超过半数都学的STEM类专业（Science, Technology, Engineering, Mathematic，既各类理工科）[2]。

[居九年](https://www.zhihu.com/people/jusjiunian)提到的竞赛与二次元联系在国内很有道理，但显然在其他国家不成立，因为大多数国家不存在小初高阶段专为升学开展的数理化生做题竞赛，不过国际上的确有一个十分类似的情况。

把二次元与STEM联系起来的似乎是1980-2000年这个阶段，这期间有这么一些情况[3]：

- 个人计算机的出现与普及，尤其是以Apple Macintosh为代表的GUI个人计算机。
- 日本电子电气产业的急速发展，以秋叶原为代表的城市在二战后因为交通枢纽的地理位置开始大量销售家用电器，并在1980年后开始转型向个人电脑[4]，这段时间内日本全球影响力骤增也得益于其电子电气产业。
- 动漫的青少年化与成人化。早期二次元的目标受众是比较低龄的，但从70年代末开始，一些日本制片人开始意识到此时是儿童的观众终将长大，动漫应陪着他们一起长大，从而开始着手于制作更加成熟深刻的内容[5]。
- 1983年互联网[6]的诞生。早期互联网的用户（显而易见的）有大量理工类工作者或理工科学生，从采样池的角度上保证了二次元对理工人的曝光度。
- 朋克文化的渗透。朋克从70年代兴起[7]，到80年代90年代时，朋克作为一种音乐流派已经开始细分，其反抗精神也从音乐步入到了生活中，正好赶上千禧辈的诞生。

这几个现象的排列组合带来了新的结果：

- 90年代，反抗精神在互联网中引导了一波二次元盗版热潮[8]，各类刑刑涩涩的种子漫网飞舞。
- 由机动战士高达为起始[9]，不断出现以青少年和成人为主要成员的线上硬核粉丝群体。粉丝群体的支持开始能对制作提供正反馈，促使创作者达到更好的成果，形成了和此前完全不同的商业模式[10]。

此时二次元与理工的联系就比较明确了，理工人对比其他人来说，**因为对技术的熟悉，比常人更早、更广、更容易的接触到二次元**；同时也是因为对技术与知识的追求，理工人比常人更容易对硬核内容产生兴趣[11]，尤其是专业对口的那种（典型例子如机动战士高达、少女与战车、魔法少女小圆）；又得益于理工类专业信仰，更容易深度投身于这个爱好。

此外，Stephen Reysen在其一书中提到，大多数二次元都是自由主义 (liberal)，因此更容易接受变化也更容易接纳团体中的新成员，也对于争议性话题更为开放[12]。同书中还提到自由主义经常对精英主义与设置门槛的行为持反对态度，这个态度和计算机领域的开源精神极其接近。如果同时考虑这两段，不难看出自由主义下的二次元粉丝群体与StackOverflow类技术论坛有着极其类似的用户画像[13]。

作业是什么？习题册要不要做？（课上说的，自愿做）

今天有虚拟实验，需要去教室吗？（上课前30min问的，答案是需要，群里老师解答了好几遍同样的问题）

看到这种问题就来气。

有人不爱爬楼去找答案，就把相同的问题再问一遍，有的时候就会得到不同的答案（第二个答案），他带着那个不同的答案心满意足地关掉聊天框；

后来愿意爬楼的人爬到两个不同的答案，心生疑惑，于是又把问题问了第三遍，也许又得到一个不同的答案（第三个答案）；

之前的人看到第三个答案，心生疑惑，又把问题确认了一遍，可能又得到第四个答案…

先把规矩立明，能减少不少的麻烦。

**很多人在提问的时候并没有意识到，自己三言两语看似随意敲下的问题也是有成本的，**成本就是群里上百个人的浏览时间和解决问题的人的精力。有些问题，比如狄利克雷函数中Qc表示什么（高数群里问出来的问题），只需要拥有一个搜索引擎和一点点搜索技巧就能解决（如图），就没必要问在群里让老师解答，这是浪费资源。

毕业多年后回想过来，我庆幸我做到了，我在恶劣无比的环境和折磨里没有提起斧头砍向任何人，也没有砍向自己，成为了一个合格的鼠人。而当去年我再次阅读《罪与罚》时，却惊出了一身冷汗，我发现我突然读懂了马尔梅拉多夫：这样一个能把自己女儿送到妓院的无可救药的醉鬼，明明内心无比渴望着救赎和改变，怀着对妻子女儿无比的愧疚和深爱，而当命运给了他最后一次机会——让他有了正经体面的工作，让他可以赎回过去犯下的一切错误之时，他却再次选择了堕落——再次喝得烂醉如泥，把工作和全家未来的希望都丢了。

学生时代一直不能理解马尔梅拉多夫，我以为是作者刻意安排的桥段。而现在我突然恍然大悟，我明白了，鼠人当久了，心就死了，哪怕怀有对家人多大的愧疚和歉意，哪怕嘴上说着希望能够改变，然而死了就是死了，成了一具只会执行机械行动的僵尸，没有了意志，没有了火焰，甚至道德都没有意义，给僵尸注入活力剂不会让他变成人。

对鼠人来说哪怕改变命运的机会扔到你面前对你来说：来，捡吧，这是你最后的机会了，他同样也会无动于衷——哪怕他明知自己只要前进一步，甚至简简单单地迈出一小步，就可以改变命运改变人生。

我终于明白了，尼采为何拥马痛哭，鼠人就是那批被无尽苦难抽打的而无法前进的马，对鼠人而言苦难和折磨是无止尽的，疯癫哲学家的眼泪根本改变不了鼠人的困境，当鞭打停止之后，鼠人也不会向前，鼠人永远会绷紧全身肌肉等待下一鞭的到来。尼采诗歌中人的三种变形——从骆驼到狮子到婴儿，根本不会发生在鼠人身上。

我控制住了自己没有成为拉斯科尔尼科夫，但我同样不想成为马尔梅拉多夫，不想成为那具僵尸，不想让机械而麻木的行为控制自己，这就是卑微鼠人最后的意志和火焰。

接下来说公费师范生。

公费师范生分两类，部属和省属。简单粗暴的说，前者分数高（在山东来讲需要达到985守门员水平，不挑专业的话基本省位次14000以前），后者分数低（普通学校二本水平也可以走提前批去上，我还见过高考只考了本科线边缘的上公费师范的学生）；前者就业面广（只要求回本省，大把学校可以挑），后者需要定向（回某个市，能去哪可能由不得你）。前者是目前形势下只想要保就业的话依然值得一试的选择，后者是你实在觉得自己上别的专业更没出路才去的选择。

学费全免，每月补贴当然是很香的，尤其对于家庭负担重的学生；但你要知道，所谓的自带编制，它可以是你的救生绳，也可以是你的铁链子。

我进了自己中意的学校，对自己的工作很满意，我是非常感谢公费师范生政策的。但我也见过有被分到了乡镇中学，想调动找不到门路，想辞职交不上违约金，想考编又是违约的老师。工作感受不到乐趣，生活也一团糟，看不到发展前景，这种时候，六年时光蹉跎就很悲剧了。

至于总有人喜欢用公费师范生分数线不断上涨来说明教师这个职业有多香，其实有点误导。部属公费师范生的分数线就没有低过，近几年我持续关注陕师大，也没有明显的上涨。省属公费的分数线确实升了，但这跟经济形势不好，人们更倾向于选择保就业的专业有关系，至于几年后到底能就什么样的业，现在还没人愿意去考虑这个问题。

说个关于私立学校的事儿吧，吾有一友系列，事儿是真的，细节别打听。

某市里的最好的私立学校（初中），一直以来都是考试选拔入学，全部费用一年下来4-8万不等。师资设施等等都一流，家长冲着名声和环境挤破头也想把孩子塞进去那种。

学校老师的收入是我们这些带高中的都眼红的水平：他们周末双休，没晚自习，节假日正常放假，然后比我们这些一个月上三十天班的拿钱还多。福利也是杠杠的。

然后忽然有一天，说是为了更好的教育公平，取消私立学校的考试选拔，改为划片+摇号结合入学。我现在带的这届高二，他们就是当初入初中的时候赶上了这一波神奇政策的倒霉孩子。他们不少人跟我聊天的时候都对这个事儿心怀怨恨，觉得他们原本有能力考上好的初中，结果被随机摇到垃圾初中去蹉跎了三年，上了高中感觉跟私立毕业的同学比起来矮一头。

再后来，经过了近两年的极限拉扯，说是为了更好的教育公平，私立学校要归公。

具体是怎么操作的，我们议论了挺多次，但谁都说不出所以然来。

然后，这个学校的老师呢就坑了。因为学校转公之后，他们这些在私立工作的是都没有编制的。所以他们一夜之间全成了合同工，工资连原来的三分之一都不到，各种福利和保障也没说怎么办。

关于求职，说一个点吧，只针对高中及以下哈，大学超出我的了解范畴。

在知乎经常刷到相关问题：我是什么什么什么条件，我家里怎么怎么样，适合当老师吗？

这种问题真意义不大，因为众所周知，所谓公务员和公务员是不一样的，老师和老师也是不一样的。

大家理想状态下的当老师：收入还可以，工作算清闲，工作环境和强度都不错，有时间顾家顾孩子，有点职业成就感和幸福感。

那至少你要做到：

入职规模比较大的重点/实验公立学校（工资发放，工作时间相对规范，学生和家长素质相对较高，工作环境相对怡人，职业稳定性相对有保障）

自身具备较好的能力（并不是你能考的上就一定能教的好，如果你备课教学或者管理学生的能力欠缺的话这个班依然上的很累很不讨好，还可能被学生和家长吿）

不当班主任，不当班主任，不当班主任（当不当班主任完全是两份工作相信我，比如我现在过得挺清闲挺快乐，如果我在我们学校当班主任那呵呵了，过不了大半年我就得保命辞职，这工作致命性极强）

有比较平和的心态（当教师的人其实卷王很多，一个组的老师主动开始内卷的时候把所有人都搞得非常疲劳，所以适度摆烂，不争评奖，不指望往上爬其实才是幸福感根源）

所以你可以看到有的人当老师收入都难以糊口，有的人年纪轻轻一身病，有的人被压榨所有的时间毫无喘息之机，都是因为他们并没达到大众眼中的“老师”的状态。

所以你如果想要选择这个职业的话，首先比较漂亮的学历和证书做个保底，然后目标明确很重要：想离家近，还是福利好，还是学生素质高，还是发展平台好，有所取舍之后再去了解你目标学校的情况。

了解学校的情况，不要亲戚朋友瞎打听，唯一可靠的信息来源只有该学校负责招人跟你联络的那个负责人，你有问题全都详细的问他。一般来说，适度画饼会有，过于离谱不会有。另外就是就职于该学校的，比较年轻的教师，他们目前的待遇和生活跟你入职后会比较贴，你想办法自己去联系。总之对学校的情况充分了解是非常必要的，不要等进了再后悔。

最后，别对自己的条件过于自信。你要知道其实大部分人都是普通人，条件不是特别出彩的话，其实大多数时间都是单位在选择你。

我忽然想起来更新了

说一个问题，关于不同学科教师的需求。

总的来讲，语数外必考主科不会产生很大变动，这几年最不缺人的是英语（不知道为什么，每年报的人都巨多巨卷），最有优势的是数学（刚需没办法，补课需求也可观，但是学生学不明白是很让人受不了的，心态要好）。

但是选考科目，对于采取3+3模式新高考的地区来讲，就有点名堂了。

不同层次的高中，对选考科目教师的需求差异特别大：因为物化主要是前列的学生在选，历地主要是后面的学生选，生政比较均匀。

高一没区别，所有学生都是九门功课。一旦分科，科任教师之间的不平衡就有了：比如我们学校就算一个年级六十个班吧，物理能选40多个班的学生，化学稍微少点，生政都是三十个左右，历地可能就能凑十来个班。

所以我们学校是有很多历史地理老师，入职很多年依然在高一年级打转转的。没办法，高二实在要不了那么多人。

NOI/IOI给了这些孩子们一个展示自己的舞台——当然，也是踩在无数鸡娃大军的尸体上脱颖而出的。每年我都主持比赛，见过了其中的人生百态。为了成就这些Gifted Ones，学校一直以来都他们一些特殊的优待，毕竟你保送上了清华北大，你就是被保护的对象，该挨的大棒都到不了竞赛生头上，不需要对老师像舔狗一样察言观色，肆无忌惮想干嘛干嘛。

**但与此同时，我们的社会其实是对这些人有所亏欠的。**

**亏欠来自于Gifted Person其实需要特别特殊的、社会更资深专业人士的正确引导**——他们或多或少有些偏执，甚至显得神经兮兮的（比如韦神）。一个极为遗憾和残酷的事实是，中学教师可以尽心尽力、鞠躬尽瘁，但难逃在这些Gifted Person面前是能力完全无法匹配的井底之蛙的现实。

1，给孩子高于平均的生活费以及一笔比较大金额的金钱，他是否就会大手大脚，花天酒地，不认真学习？

先说答案，我觉得不会。这些资金的用途我都说过了就不再赘述，首先父母愿意给满18岁的孩子这笔钱是承认了他的成长和独立，不再把他当做一个什么都不懂的控制不了自己的儿童，这是父母对孩子的信任和尊重。根据马斯洛需求理论，五个层次生理需求，安全感，社交需求，尊重需求，自我实现，当孩子得到了前三项，同时有父母的尊重和支持，他就进入了最高需求阶段自我实现追寻梦想，那么在大学里就会专注于目标，不管是学业还是社交甚至是爱好，从而成为将来社会上立身之本。另外一方面是否乱花钱我也不是很担心，从我自身的体验来看，对于物欲最渴望的就是人想要而得不到的时候，反而当它唾手可得的时候对你的吸引力就不那么大了，例如说一个LV包包，一个学生或者刚毕业的小姑娘可能会为了它节衣缩食，但当你一个月工资可以买两三个，也许你会发现这也就是个可有可无的选择罢了。如果少给钱就能控制孩子的物质欲望吗？不见得，很多大学生网贷的新闻爆出来，最开始也只不过是为了一双aj，一个苹果手机，一台手提电脑，结果最后陷入泥潭不能自拔，甚至闹出更大的悲剧，所以适当的物质满足是必要的。

当然如果家长只给钱，不引导孩子使用和储蓄，不检查是否用在正途，那也是不行的，毕竟暴富之下也有可能冲昏头脑。

2，富养比穷养来说，到底有什么好处？

最直观的说，富养的孩子普遍比穷养的孩子自信程度更高，他们相信自己配得上更好的大学，配得上更好的爱情，配得上更好的工作，也配得上更好的人生。既然社会上年薪几十万上百万的高薪工作是真实存在的，那为什么不能是我？只要我努力这一切都有可能发生。

反观有些同学的发言，说现在给三四千，万一工作以后只能挣三千养不活自己，所以现在少花点，学着习惯一两千的日子，这真的大概是另一种思维惯性，我不知道怎么形容…………如果你自己都觉得自己不行，那就真的没有人相信你行了。

以上就是我建议家长在自己家庭经济条件的基础上尽量按照略微宽松的标准给孩子大学生活费的原因，希望大家理性探讨。

时隔一年，来给大家汇报一下最新的生活费用情况：

一，生活费用从每月3000涨到4000，衣服报销额度不变，也就是说每月平均大概5000，原因一是现在物价确实有所上涨，二是随着恋爱的进程开销也增加了。

二，理财计划宣告破产，这一年花了三万块，其中买了个iPad用来上课记笔记，另外因为加入了车协，买了一辆2万左右的公路单车，另外还有几千零零散散的花掉了。剩下的钱儿子还给我了，让我帮他存着，如果想用再跟我商量，他觉得自己拿着总有想花钱的冲动，现在只能做到的是量入为出。我开玩笑说那以后工作了是准备月光吗？他说看来只能结婚前把钱存在我这，结婚后交给老婆存着。

看来是不是女生天性比男孩更有计划，更会存钱？还是遗传使然，毕竟他爸爸好像也是这种存不住钱的主。

割裂感，一种极强的割裂感，一种史无前例的割裂感。

现在的年轻人，我大致定义在1995到2005年左右出生的人吧。

这段时期出生的人，无论是哪个阶层，无论是城市还是农村，他们都有着极其强烈的割裂感。

先是科技的割裂感：我一直认为1990年开始，科技进入了一个飞速发展的时代。

2000年的时候，电脑，还没有普及，偏远一点的县城，农村很多人甚至都不知道电脑是什么，到了2006年左右的时候，电脑已经很普及了，县城有了很多家网吧，就是消息闭塞的，没有什么文化的农村老人也知道什么是电脑了。到了2013年的时候，二维码，智能手机，移动支付已经在大城市普及，而这时候很多县城的人大多数还没有智能手机，这可以说是科技和城乡双重的割裂感。

再之后多屏协同，物联网技术，大数据，区块链，人工智能…这些在十年前我们都无法想象的东西都有了。还记得小学的时候有个作文，让大家想象未来的科技，结果现在回头看，当时很多异想天开的内容都实现了！

这种科技的割裂感是绝无仅有的。

你让一个汉朝的人穿越到一千年后的宋朝，他可能比较好奇，但是他不会惊讶。但是最近十多年的科技发展足以让人惊叹。

我是东北一个小县城出来的人，那个县城很闭塞，与大城市都慢半拍。我还记得大概在2010年的时候，我们那里的小学弄了个“校讯通服务”就是家长花钱，之后学校的信息，比如说哪天开学，学校组织扫雪活动，还有平时的作业什么的，都会通过手机短信的方式发给家长。到了2014年，智能手机普及的时候，觉得当时的校讯通是多么的荒唐啊。而2013年大城市的人，也会觉得所谓的校讯通很荒唐吧。

还有教育的割裂，上小学的时候，老师教育我们，要为集体做贡献，舍己为人。后来发现，我们上学只是为了取得更高的成绩，去干掉自己的同学，只有成绩好的学生才能得到家长，老师，七大姑八大姨的认可，才能上初中的重点班。

七八十年代出生的人，如果他参加过高考，他会发现，高考就是书本上讲什么，就考什么。而之后的高考题目越出越活，如同变戏法一样，只学教科书一点用也没有，大量教辅书涌现，这种教学与考试的割裂感也是绝无仅有的。

还有自身感受与主流思想的割裂。奥运会，冬奥会，我们跟着赞扬运动健儿，跟着主流思想说吉祥物好可爱（可能你根本不觉得吉祥物有多么可爱）国足输了，不仔细分析内部底层逻辑，但是主流思想认为是国足不好好训练，你跟着怒骂国足。电视上宣传高考状元，家长要你和他们学习，你总感觉怪怪的，但是还是点头答应（就县城那教育资源，学习氛围，和人家状元能比吗），再后来，写作文，要写家乡的变化，你按着辅导书上的思路，说家乡变化大，以前吃不饱饭，现在家家户户都过上了幸福的生活。平时班主任为了管理班级方便，也传输了很多错误思想：比如什么一个巴掌拍不响之类的。还有，学校那么差，不还有考好的学生嘛。有时候班主任为了赚钱，私自补课，排挤不去他那补课的学生，你是少数没去他那补课的学生，因为你特殊，你没去老师那里补课，你总觉得自己被集体排斥，同时，大多数同学都去补课，你又怕落下功课，觉得自己做错了什么。平时老师，家长教你诚实守信，正直善良，但是教育局查补课的时候，老师总是叫你们蒙骗教育局的工作人员。你的父母，长辈明面上面对亲友，和颜悦色，背后里经常议论邻居，亲友的琐事，诸如：张三随礼随少了，孩子他二叔压岁钱给少了，谁家孩子学习不好，一看就是笨…你和家长说，你平时教我诚实，但是你自己却那么虚伪，这不是矛盾教育嘛，你的家长无法反驳，但是仍然对你一顿痛骂。上小学的时候写作文写自己的家乡，总是写自己的家乡多么多么好，但是你的家长，老师总是说让你好好学习，去大城市，脱离这个小地方，脱离这些低素质的人吧。你的家长刷小视频，当时美国列出制裁中国高校名单，有很多军工院校。小视频的制作者为了吸引流量，故意用“华为最喜欢从这些学校招人”，或者是，“清华北大都去美国了，家长爱国就要送孩子去这些学校”这样的标题吸引眼球。殊不知，这些学校大多数都是行业类高校，上不了它的核心专业，性价比很低，你的家长和你推荐这些学校，你知道是怎么回事，却不敢反驳，因为如果你反驳，一场无意义，内耗的争吵就开始了…有时候你感觉这些主流思想是不对的，但是面对那么大的舆论压力，你只好做造成沉默的螺旋这种现象中的一员。有时候，你忍不住公开反驳这些主流思想，但是很快，周边人就开始对你排挤，攻击你，你最后不得不妥协，否则你会变成大家眼中的异类，于是你跟着大家骂国足，跟着说吉祥物好可爱……最后你完全麻木了。再后来，你上了大学，有了疫情，很多家长老师，都说要考研，要去国企，有稳定工作…再后来你回过头来看以前每一段时光你面对的主流思想，你发现，很多事要辩证地思考，有时候就是那段时间的一些主流思想让你走了错路。其实当时你也觉得怪怪的，但是，以你当时的心智，无法独立思考。







甚至有的时候，即使是所谓的主流思想，也是割裂的。大家一面称赞大国工匠，说他们说国之重器，称赞扎根乡村的教师，称赞不慕名利的科学家，称赞驻守一线的军人。同时每次看完《感动中国十大人物》老师让你写读后感。但是，你的老师，家长却告诉你，读书要成为人上人，看到环卫工人，你的家长告诉你，你不好好学习，只能成为他那样的人，而不是说你要好好学习让环卫工人有一个更好的工作，生活条件。平日里，学校经常播放记录片，讲述革命家，烈士们的革命生涯，让学生写观后感，观后感也大多是写：努力学习，报效祖国，不辜负烈士们的牺牲。你们村里的老人也经常说，没有烈士们的牺牲哪有今天的幸福生活。但是另一面，村里老人重男轻女，有很多封建陋习，愚昧思想，老师也总是教导你们要做人上人，当工人，农民是可耻的。很多人都说：要努力学习，工作，不辜负革命家和烈士们的奉献和牺牲，这句话基本上成为了观看革命题材的记录片观后感的标配。但是，我们发现我们努力学习，工作，就是为了成为老师口中所谓的人上人，很多人嘴上说不辜负烈士们的付出，心里却有很多落后封建思想……每年清明节，学校都要组织班会来表达对烈士们的哀思，但是班会的下一节课，老师就告诉我们要成为人上人，难道烈士们用生命换来我们今天美好的生活就是为了让我们奋斗，做所谓的人上人，不去做老师口中所谓的底层人：工人，农民吗？记得小学时老师也总是偏向家里富裕，学习好的学生……真不知道烈士们的在天之灵看到这些会作何感想，或许烈士们看到这个社会也会感到割裂吧……有时候看着那些虚伪的人在烈士墓前，革命家的雕像前献花，感觉是对那些烈士和革命家的侮辱。现在回想起教员上学时的宏图大志和无私奉献，总理当年为中华之崛起而读书，愧疚难当………

还有各种信息给你造成的割裂：抖音，快手，一群富二代让你误以为有钱人很多，知乎人均985让你误以为上个985很轻松。再回头看你们县城：一个月能赚个五千都是高收入人群，县一中第一才能勉强上个985。再后来你大学毕业感觉工作难找，看你的同学一个个光鲜亮丽。

还有自身人际关系的割裂：你出生在一个县城或者农村。小时候，你不知道为什么，你正常做作业，正常学习就能考出很好的成绩，有时候你觉得自己成绩太好，有点不合群，你可能有点羡慕那些中等生，觉得他们合群，有归属感。上了初中，学业压力变大，你成绩也不错，上了高中，你发现自己的分数上不了什么太好的学校，但是在这所高中排名也挺靠前，高中同学叫你学霸。同时，你感觉和身边的同学仿佛本质上成为了敌人：在东亚内卷文化体系下，你要通过竞争掉你的对手，也就是你的同学，才能获得世俗意义上的成功。你的内心厌恶这种世俗的价值观，世俗的体系，但是你父母口中别人家的孩子，你父母和身边亲友的攀比，你要努力学习，为自己以后优质的物质生活而奋斗，又使你不得不卷入这个体系中，你觉得东亚内卷文化思想反人类，你希望的事是人文主义，社会主义思想，但是这些仿佛离你很远。你怀着复杂又不适的情绪进行着你的学业。

在日常生活中，你发现你和你的同学格格不入：你的同学总是在一起聚餐，谈论游戏，而你专心学习。你和你的亲戚也格格不入，你觉得他们素质低下，没有文化，愚昧无知，你的亲戚认为你肩不能扛，手不能提，百无一用，（虽然他们明面上没说，但是通过他们那低贱猥琐的表情和一些话语你能感受到他们的心理状态。）当你最后没考上清华北大的时候更加深了他们对于他们下的结论的正确性（对于那些没有文化，素质低下的人来说，除了清华北大，剩下的学校都差不多）同时，他们通过自己仅有的那一点获取信息的渠道：电视上的新闻，街头巷尾的留言，和刷小视频赚钱的软件，以及充斥着类似于“惊！某人说出内幕，快点转发”这样推送的软件，知道了大学扩招，不好找工作，所以在他们的认知里，大学除了清华北大，剩下的毕业了都很难找工作。即使你上了985…他们会认为西安交通大学，吉林大学这样的学校不如你们省会的二本…

再后来你高考考上了个普通二本，但是名字叫的挺好听，叫某某大学。身边的亲友也没什么文化，只知道清华北大，什么985，211这个专业，那个专业什么都不知道。只是觉得你的学校名字挺好听，都随声附和，这孩子，有出息了。你在回过头来看自己小时候的伙伴，同学，他们中的大多数要么初中毕业在社会游荡，要么职中毕业上大专。你如同是一支突围部队里仅存的一个战士，你这时候有点沾沾自喜，同时，也很悲怆，一种前所未有的孤独感笼罩在你的心头。

你的大学同学中总有那么几个家境好的：大城市有好几套房，家里做生意的，或者说你中学的时候身边就有同学家境不错，靠家里的房产收租就能生活。你上大学的某个课间，你的暗恋对象问了你一道高数题，你是班里的小学霸，你给她解开了疑惑，她很高兴，向你投来了崇拜的眼神，你傻乎乎地笑了。但是没过多久，你看见你的暗恋对象和你们班的富二代牵手了，你的心中的自卑感，落差感…各种感觉交织在一起，五味杂陈。实际上，你小学，中学就曾经有过这种感觉，还是那个课间，你暗恋的人问了你一道数学题，你很快得出答案。但是她却转身，投向了班里富二代的怀抱……你从小学就开始质疑家长和你灌输的知识改变命运的话，你坐在教室，看着窗外，看着这个物欲横流的社会，看着自己暗恋的对象投入富二代的怀抱，你感觉自己即使努力学习，也很难抹平这么大的差距。

你大学毕业，你的同学不知道是装的，还是家里有资源，一个个去向都很好。

只有你找不到合适的工作，在外打拼。

过年了，你也不愿意回家，因为一回家，那些没有文化，素质低下的亲戚都来嘘寒问暖，说着虚伪的话，各种与你三观不合的观念从四面八方如同潮水般向你袭来，你无力躲开。再看那些以前不如你学习好的同学，很多都已经成家立业，孩子都有好几个了，自己觉得惭愧，不敢面对以前的同学。

同时，你的父母与你也经常吵架，因为各种观念不合…

而且割裂感不是把一个事一刀切为两半，而是把它切碎，切得极碎。比如说建设家乡这个事：社会主流观点：支持乡村，县城振兴仿佛成为了政治正确。但是大家私下里都认为留在家乡的人没出息。你觉得身边的人，包括老师，家长虚伪，你内心的道德狠狠地鄙视他们虚伪，但是你一想到家乡贫瘠的文化环境，物质条件，想到素质低下的身边人，你又想去大城市拼搏…但是语文试卷上出现了大学生回村创业，大学生当村官，你知道他们回村只是把村子当跳板，或者有利可图，不是为了建设家乡，但是你为了得分，仍然言不由衷地写下了所谓的正确的答案。看吧，就是一个小事，就有如此多的观点在激烈地碰撞，为了获得利益（分数，大家的肯定）你来回地转换着自己的观点，最后你都忘了自己最初的理想是什么……

这一个个割裂感，如同是一把无形的刀，分割着现在的年轻人，让一个个年轻人生活在一个个孤岛上，同时，这些在孤岛上的年轻人，又每一天都面临着割裂感：这割裂感来自于优秀的别人家的孩子，来自于以前学习不好，但是已经成家立业的老同学，来自于电视上一切向好，稳步向前的就业形势……甚至有时候，割裂感间接地来自于你自己。你看着知乎上的人均985和抖音上一群富二代，想着自己小时候的理想，面对着现在的处境，再想起以前学习不如你的同学，亲友现在已经成家立业，过年回家乡时他和他家长对你那嘲讽的嘴脸如同愤怒的小鸟通关失败后猪的笑容（至少你是这么认为的）你心有不甘，顿时爆发小宇宙，要奋勇拼搏。

但是不久的之后，你遭受了社会多方面的捶打：劳累的学习，工作，各种困难……你筋疲力尽，又让你觉得，能结婚成家，有个孩子，有个房子，有稳定的收入已经是奢望。你发现这时候的你心情很低落，前所未有的低落。你再回顾自己的一生：仿佛也没有什么特别值得高兴的事情。无非就是自己某次考试成绩比自己的伙伴高（但是也没什么用，因为你也没有考上什么好大学。）或者是小时候吃到自己喜欢的零食，自己受到老师的表扬。再或者是你自己小时候考了一百分，在思索以后是上清华还是北大的快乐的“烦恼”，抑或是小学时你和你暗恋的对象做了同桌，或者你暗恋的对象多和你说了几句话，你自作多情认为她喜欢你，或者是快高考了，你的分数上不了什么好大学，但是看着比你成绩还差的同学，你偷偷地幸灾乐祸，五十步笑百步，实际上你从小到大可能根本没有真正地感受到快乐，那些所谓的快乐只不过是你在没有直面生存压力下，以及你的人生没有被大范围局限下，你在压力比较小的情况下被某些现象触发的意淫罢了。难道不是吗？你没有生活的压力，你可以畅想自己未来富可敌国。你暂时没有面对高考，你可以畅想自己考上名校，同时你的同学没有你考的好，更加深了这种意淫。让你有了一种很大的快感。实际上，我觉得，小时候认为自己肯定能考上清华北大，与看小电影的快感类似，都是在暂时没有直面压力下，通过某些方式刺激自己的大脑的意淫罢了（比如说自己考得比同学好，以及小电影的画面…）但是，高考这座分水岭，也是大山把这所谓的快乐压碎。高考后的你，没有了意淫自己考上名校的快乐，而且你的未来大概率被局限在某个区间，同时你要直面为了生存的压力，你发现，自己好像从来没有过快乐，以前的快乐，不过是一个幻境罢了…

在某个夜深人静的时候，或者某个闲暇的时候，你突然回想起自己经历过的某个事，再根据这个事联想到其他事，最后以点连线，以线带面：你会发现，从小到大，你接受到的大多数信息，很多都是假的…可能你考上大学，亲戚恭喜祝福你，很可能他们因为自己的孩子没你考的好，背后嘟囔，不就是个普通大学嘛，有什么好张扬的，你的混混同学表面上叫你学霸，背后里很可能觉得你是个书呆子，不会为人处事，以后吃不开，而且你的混混同学也知道，你的成绩也上不了什么好大学，只是在成绩很差的县一中里相对好一点罢了，但是明面上还是叫你学霸……你发现你从小到大，几乎每一个人和你说的话都是虚伪的，都在和你玩心眼，但是你现在才发觉出来，或者说你以前看着亲戚那虚伪的笑容，混子同学那阴阳怪气的语调…感到不适，但是这时候的你，才彻底地发现，自己一直被骗，一直在别人设的局里…你发现，自己以前如同是一个被人愚弄的傻子。同时，你发现以前的时候，你身边的人，乃至于你的父母，还会通过一些“巧妙”的方式来使你图痛苦，从而实现某些目的，或者说单纯地通过使你痛苦来实现某种快感。你的父母通过别人家的孩子，让你无力反驳，因为真正侮辱你的，是别人家的孩子，你总不能把别人家的孩子无缘无故揍一顿吧…甲看你不顺眼，和你说：“难怪乙说你哪里哪里不好，太差劲了。”你无法争辩，因为侮辱你的主体是乙，不是甲，你发现即使这是一个法制社会，身边的人依旧能用无数种方式来恶心你，使你陷入无尽的精神内耗……你从小到大可能收到过很多这样的内耗，（我们那里把这个叫“踹股”）但是你却无力还击，只能一直陷入这种精神内耗，可能你的学习成绩差也是源于此吧…

而你现在的困境也无处宣泄，这里面有你的物质困境，也有你的精神困境。你和别人说了，别人或许明面上安慰你，背后对你也是无尽的嘲讽。

但是，你又往深层思考：那些和你玩心眼，内心嘲笑你的亲戚现在过得也不怎么样，你那些以为自己多么厉害，多么聪明，觉得你不会为人处世，情商低的同学现在也不比你好到哪里去……他们把精力很大一块放在了琢磨别人是多么的愚蠢，来证实自己多么的聪明，从而实现了阿Q式的精神胜利，底层人的内耗就是这么的严重，都认为别人是傻子，自己有多精…实际上生活中绝大多数的事都是小事，做错了也没什么，但是很多人就喜欢揪着这些小事不放，明里暗里笑话别人，而真正重要的就是那几件有关于未来方向的大事，没几个人有能力掌握，或许那些人也知道自己没那个能力把握大事，只能从对别人小事上遇到麻烦而幸灾乐祸，从而获得低级无味庸俗的快感吧……有的时候，你也扮演着幸灾乐祸的角色：你在一所普通大学，你知道自己的未来大概率局限在某个区间，但是你用自己考了很多证来麻痹自己，不去认真思考未来的路怎么走，并且通过对于你的大学同学没有拿到这些证，或者你的同学因为一些事受挫的幸灾乐祸来加大这种麻醉……其实，这个世界上有能真正能活明白，看明白事的人，太少了……

同时你也对于你所在的圈子感到深深的绝望，你觉得你所在的圈子仿佛是一个低矮的猪圈，里面有一群猪在抢着粗劣的食物（如同社会中下层，或者说对标普通大学，各种资源质量少又差了，而且竞争者多）有的猪因为抢到了更多的稻糠而沾沾自喜，有的猪因为看见别的猪摔倒了而幸灾乐祸，你也是其中的一头猪，你想跳出这个圈子，但是，你往出跳的时候，其他猪有的嘲讽你，有的甚至在妨碍你往出走。

你回想起你自己的人生，很多时候难以有突破，也是被这一双无形的大手限制住了，在县一中读书很难考上好大学，在普通大学很难找到好工作都是这双无形的手的杰作，就如同是一个筐里放着一群螃蟹，你是那只想往上爬的螃蟹，但是身边的其他螃蟹要么拽你的腿，要么压着你，不让你往上爬，如果在那个筐里的只有你这一只螃蟹，你早就爬出那个筐了，这时候，渔夫把这个筐里的所有螃蟹一起放到锅里清蒸，这时候的你愤恨，后悔。你所在的筐就如同你从小到大身边的生存环境，学习环境，亲友，同学圈子，你从小到大学习成长学习的过程如同往上爬的过程，而你最后面临马上被清蒸的境遇如同马上面临社会险恶的你……

你发现从小到大没有一个可以交心的人，同时你所在的圈子一直在内耗，大多数人内心都在算计自己，你感到以孤独为主体的感觉，而且不止是孤独，还有矛盾，有对于未来的恐惧，以及对于之前某些行为，思想的悔恨，这就是割裂感。

割裂感把你分割在一个孤岛里，同时，这个孤岛里的一草一木都有着割裂感，甚至空气里都弥漫着割裂感，让你体无完肤。

至于生活压力大之类的。其实你看看历史，历史上很少有现在这种和平稳定，人人都能吃饱的年代了。历史上大多数的年代都伴随着饥荒，战乱……

但是这种割裂感却是历史上绝无仅有的。

后记：

这篇文章是我在2022年5月份写的，2022年5月初某天傍晚，我在知乎上看到了这个问题：“为什么现在的年轻人越来越悲观。”自己刚开始看到这个问题，内心突然有了一种前所未有的触动，想写一些东西，之后看了其他答主的文章，也产生了很多的共鸣。于是结合自己和自己身边亲友的成长，学习，工作经历，洋洋洒洒地写了这篇文章，写完之后大有一吐为快之感，仿佛把积郁在胸中许久那股难以形容的郁闷之气吐露出来，顿时感觉前所未有的畅快。之后又根据自己的回忆，断断续续地想到了以前的一些经历，再加上各位朋友陆续的评论回复，以及私信，很大程度上启发了我的思想。因此写完文章之后又不断对这篇文章进行了完善，补充。这篇文章能获得这么多的赞同，引起这么大的反响和共鸣也离不开大家对我的帮助，感谢评论区以及和我私信的朋友，感谢你们宝贵的意见和支持。

同时，我看了评论区，以及有朋友发给我的私信，一些事情又引起了我的思考：产生我前面文章提到的现象的底层逻辑是什么？我思考许久，我认为产生割裂思想的根本原因是因为：我们国家改革开放后的几十年到现在有三种底层逻辑：1：社会主义集体化思想，2：中国传统忠孝思想，3：资本主义功利思想。这三种思想有互相矛盾的地方，有的地方比较相似，（但不是相近，更不是相同，相通）这三种思想组成了我们社会舆论，主流思想，以及人们世俗的，功利的思想的底层逻辑，我们身边的一些社会现象，一些让我们感到割裂的意识形态，一些和我们让我们感到拧巴，感到不适的思想，文化氛围，很大程度上是不能调节好这三种思想的关系，而产生的现象。

这里强调一点，我之所以用中国传统思想而不用中国儒家传统思想来形容我们国人特有的文化特性是因为我觉得儒家思想具体讲了什么，我们现代人很难去还原。儒家的创始人孔子去世之后，儒家分成了八派，每一派的掌门人都是孔子的弟子和子孙，而且都声称自己是儒家正统，试想一下，这些人活着的时候和孔子同吃同住，这些人都不知道孔子的思想具体是什么，我们又怎么能大谈孔子思想呢？更不能冒昧地认为所谓的孔子思想害了国人。因此我没有在中国传统思想里加了“儒家”二字。

我前面提到的三种思想，以及这三种思想的交汇融合和矛盾之处构建了我们的精神世界。比如老师宣传为集体做贡献是社会主义集体思想的体现，家长背后议论亲友，互相算计，亲戚无边界地打探隐私，攀比，则是我国传统思想的体现，社会风气功利化，缺乏职业道德，嫌贫爱富则与资本主义思想有关。电视里的宣传片中提到党员干部的家风家训则是中国传统思想与社会主义集体思想的融合…

同时，我们国家的传统思想本身也有矛盾之处，我们的传统思想表面上教育和睦谦让，孔融让梨，但是历史上，明争暗斗，甚至弑父杀兄的故事不断，而争强好胜者往往也是既得利益者（李世民，朱棣，还有五代十国类似的例子不胜枚举）。传统思想表面上教育忠君爱国，宣传天下兴亡，匹夫有责，“苟利国家生死以，岂因祸福避趋之”，但是王朝灭亡之际，绝大多数文臣武将却各怀鬼胎，各谋出路。更有甚者，打着匡扶国家的旗号行利己之事，两头下注，左右逢源的例子更是不胜枚举。一边宣传君权神授，一边历史无数次验证了：“天子者，兵强马壮者为之 ”的客观事实。一边宣传文天祥，岳飞的爱国节操，但是更多的则是类似于耶律德光入侵中原之后中原各镇望风而降。五代十国时期社会动荡，天下大乱，则是上述极端情况的具体表现（欧阳修说：五代无全臣）当然，以上现象也体现了现代化国家概念的普及和集体主义教育，与爱国主义教育的重要性，同时利用中国传统文化宣传天下兴亡，匹夫有责之类的古语来强化爱国主义教育从某种意义上也是社会主义集体主义思想和中国传统思想的融合。

而这三种思想之间的矛盾和尴尬，令人不适的融合，各自本身的矛盾之处则是让我们感到割裂感，产生不适的根本原因，我记得小学初中的时候，有一门课叫思想品德课，小学时候这门课是水课，不考试，初中这门课就是背书，考试类似于默写背书的内容。当时觉得思想品德课一是没有意义浪费时间，背诵记忆那些无用的内容，二是觉得思想品德课教育的内容本身也空洞与实际相违背，三是觉得思想品德课的内容一堆仁义道德好虚伪，授课老师都不相信那一套，只是为了完成自己的工作罢了。思想品德课好像是老师在骗学生，学生也在配合老师的演出，空气中弥漫着虚伪的气息。为此当时的我肤浅地认为思想品德课的本身是毫无意义的。（我记得初中也叫它政治课）后来我才明白，政治的内容很深奥，并不是满纸毫无意义写满仁义道德的书籍，而是讲述了社会的运行规律，资源的分配规律，包罗万象，是最重要的一门课，也是我们每一个人都要经历的一门课。

至于我觉得以前思想道德课那满纸的仁义道德是虚伪的，行不通的，也是社会主义集体道德思想与我们国家现实生活中的传统的功利思想和资本主义思想之间的矛盾的具体表现。小时候觉得思想品德课，政治课这种课是个人都能教这门课，就是照本宣科，画题背题，后来明白，能把这门课讲好，讲明白比讲好数学课物理课难多了……

当然了，我提到的这些思想实际上在很多文学作品中都有涉及，比如都梁老师写的《亮剑》后半段，我军将领李云龙，丁伟和知识分子田墨轩对于阶级问题，以及中国未来的社会建设，意识形态等问题的争论与交流，以及李云龙排队时候与他人的冲突，还有李云龙和老战友赵刚对于建国后一些现象的交谈，都或多或少地反应了我前面说的现象，还有《遥远的救世主》中丁元英提到的思想问题，都对于我前面说的思想问题有所涉及。这些文学作品的作者们借笔下人物的口吻来述说了对于一些事，一些现象的思考。

![](https://picx.zhimg.com/v2-706d135ed5b982b560a75d570d3d4992_r.jpg?source=2c26e567)

## 1. 动机

我最初看到这个问题，是因为我点进了这个问题下的一个高赞回答。随后，我又阅览了另外几个高赞回答。每个高赞的回答都有不同的特点，有些真的能唤起人的共鸣（我也是00后），有些则很有趣，有些则打动人心，有些能引人深思。

我相信大多数看到这个问题的人和我是一样的，我们都被高赞回答吸引而来，并且只看到了一些高赞的回答。然而，高赞答案不过是寥寥几十个，然而世界上有那么多的00后，我相信，这些高赞仅仅是广大00后群体的冰山一角。

那么，有意思的问题来了：**00后，作为一个广大的群体，他们的忧虑到底都有哪些？**

**解决这个问题的一个思路，就是分析更多的样本。**高赞答案不过是几十个，然而，这个问题下总共有一万多个回答。虽然这一万多个回答跟广大的00后群体相比依然只是沧海一粟，然而我认为如果我们能够分析这一万多个回答，我们或许可以对00后群体有更清晰的认知。

因此，我收集了北京时间2月6日下午3点之前所有回答的数据。数据获取过程中有极小量数据丢失(约1%)，最后我获得了**13619**个回答。我决定利用这13619个回答，来探索00后群体到底在忧虑什么。

## 2. 初步分析

首先，先看看答主们写了多少。我把这**13619**个回答按照字数的数量级归类，画出了直方图和饼图。

我们发现大部分答主的回答在10-1000字的区间，而100-1000字的答主占了超过总数一半的回答。1000字以上的回答也不少，超过了10%，**说明至少在字数层面上，有不少答主是在很用心的写作的**。如果我们认为超过100字的回答算比较认真的回答的话，那么有超过60%的答主都在认真回答问题。

随后看一下点赞的情况。我们并不意外地发现：**高赞回答仅仅占据了全部回答的极小部分**。这进一步也说明，高赞回答仅仅是冰山一角，这数十个高赞答案背后，是上万个不容易被大家看到的小于10赞的答案。

![](https://pic1.zhimg.com/50/v2-cd3f202a0b4a890f14b927f748141c6d_720w.jpg?source=2c26e567)

我们接下来可以看看字数和点赞数的相关性。凭直觉我们会觉得字数越多应该点赞越多，但其实这两个关系其实不是很强。

![](https://picx.zhimg.com/50/v2-33a8fb438f5beaaf1714c0c7b510e448_720w.jpg?source=2c26e567)

右上角的回答：[00后在忧虑什么？](https://www.zhihu.com/question/393450972/answer/1233601696)

这个回答下有很多答主在倾述自己的忧虑。所有回答共计写了数百万字，然而，它们之中的绝大部分并不是高赞，想必这些内容也不容易被别人看到。我们容易看到的高赞（1000+赞）回答，从文本量上来看，仅仅占据这全部文本的2%。

**因此，我们有必要想办法分析这一万多篇没有很多赞的回答，去分析这不容易被看到的几百万字，去挖掘冰山在水下潜藏的巨大山体。**

## 3. 答主年龄分析

在分析这几万篇共计几百万字的回答之前，我们先有必要分析一下这些回答的作者。

“00后”这个词一般是指00-09年出生的人。但是，我们能感觉出来，这个回答下的各个00后的年龄并不是均匀分布的，所以我们可以先研究一下答主们的年龄分布。

有很多答主在回答的时候都会提到自己的出生年份。因此，我们可以用正则表达式来匹配这些信息。

([09][0-9])(年|的|高中|大学|出生|生|女|男)

这个正则表达式表示我们想要匹配形如XYZ的内容，其中X为0或9，Y为任意数字，Z为“年”、“的”、“高中”、“大学”、“出生”、“女”、“男”。

这个匹配方式可能会有一些误判，但是基本还是准确的。

匹配之后，我们发现总共有**3498个**回答暴露了自己的出生年份。把数据可视化，我们能看到：

![](https://pic1.zhimg.com/50/v2-4b75b6d63c38d45bf32f514c534f9237_720w.jpg?source=2c26e567)

**可以看到，这个分布是非常倾斜的。00-09年出生的人中，出生越早的答主越多。**不过也可以理解，毕竟目前08、09的孩子们还太小。还有99年生的人回答得也不少，我觉得也是可以理解的。99年和00年、01年的也没什么本质区别。

所以我认为，**到目前为止这个问题下的回答所反映的情况，更多还是99-06年生的人在目前的情况，尤其是00-02年生的人的情况。**

我们用这些答主的答题年份，减去他们的出生年份，便可以估算出他们回答问题的时候的年龄。

![](https://pic1.zhimg.com/50/v2-9805bd588027cd76144810cac85cb3d9_720w.jpg?source=2c26e567)

**我们可以看到，大多数答主在回答问题的时候是在20岁左右。**其中20岁的人最多，15-19岁的也有很多，而且年龄越大则越多。其中21岁也有一些，22岁以上的很少。**可以看出，答主们的年纪主要还是大学生年纪，以及少部分的高中生年纪，还有极少量的其它。**

## 4. 文本分析—00后们的忧虑是什么？

如何了解广大的00后群体到底在忧虑什么呢？

我们可以有两个思路，第一种是高频词——统计每个词在所有回答中的出现次数。这种统计非常直接，也能很好的反应一个概念在这个问题下被提到的次数。这种统计方式或许能大致可以反应00后最忧虑的事物。第二个思路就是统计每个词在多少个回答之中出现，这种统计方式大致可以反应被最多的00后关心的事物。

我们使用spacy的[zh_core_web_lg](https://link.zhihu.com/?target=https%3A//spacy.io/models/zh%23zh_core_web_lg)进行分词。统计的时候要去掉各种虚词，比如的、地、得这些（严格来说是stopwords）。

> 按下键盘上 Shift + F10 的组合键，启动命令提示符。
> 在弹出命令提示符的窗口中，输入命令OOBE\BypassNRO并回车。

这一步骤确实有效，但在连命令行都一窍不通的 Windows 小白用户的眼中，如何使用命令提示符着实要花费一定的时间和精力。

此外，剖析该命令背后的实现就能发现，这一操作执行的其实是如下路径的脚本：

C:\Windows\System32\oobe\BypassNRO.cmd

更进一步，用资源管理器定位到该路径，用记事本打开脚本BypassNRO.cmd，可以看到其中的内容如下：

@echo off
reg add HKLM\SOFTWARE\Microsoft\Windows\CurrentVersion\OOBE /v BypassNRO /t REG_DWORD /d 1 /f
shutdown /r /t 0

这三行命令的作用分别是：

- 让脚本不显示正在执行的命令及执行结果；
- 向 HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows\CurrentVersion\OOBE 这一路径添加一个注册表项。其中该注册表项的键值为 BypassNRO， 种类为 DWORD 值，数据为十六进制 1，以强制覆盖的形式添加；
- 立即重新启动 Windows。

由此可见，在家庭（单语言）版和专业版的 Windows 11，为了跳过强制用户登录 Microsoft 帐户，**竟然需要在注册表上动一点手脚！**

所以问题来了，为什么微软要在普通用户身上花这份心思呢？

我不知道题主是怎么理解的，反正我的理解是：Windows 团队觉得消费者版（家庭版、家庭单语言版和专业版）用户都像软柿子一般好拿捏。

毕竟我知道，作为微软桌面操作系统的死对头，苹果的 macOS 一直都是不强制用户登录 Apple ID 的（包括今年发布的 macOS 26 Tahoe），在设置助理中会提供“稍后设置”的选项，可以跳过登录 Apple ID 这一步：

![](https://pic1.zhimg.com/v2-3e4d89f1cf65b4ae7a79384896ce5e34_r.jpg?source=2c26e567)

更讽刺的是，企业版（含 IoT 企业版）的 Windows 11，即便电脑不能联网，**在 OOBE 阶段仍保留了可以跳过联网的选项。**

这是因为在企业中，部署操作系统完全可以在不接入外网的环境下进行，而企业客户是微软绝对不能丢掉的一块肥肉。如果企业版 Windows 11 还要强制登录 Microsoft 帐户，不被客户骂算够给微软面子了。

![](https://pic1.zhimg.com/v2-84915ea9ad2267dcdcd67dea2d9e22d8_r.jpg?source=2c26e567)

当然，在内地的电脑市场，PC 经销商有一个“心照不宣”的规则：为了降低二次销售的影响，联网激活的 PC 不能享受七天无理由退货。因而，强制联网登录 Microsoft 帐户的机制对此也是一个“神助攻”，这也间接地损害了内地 PC 消费者的权益。

最后评一评题主问题描述里的这句：

> 那些骂微软的网友用手机时不可能登录的本地账户吧

这是一个典型的逻辑滑坡谬误。

首先，移动操作系统和桌面操作系统之间不具备可比性，“批评微软强制要求用户桌面端登录 Microsoft 帐户”与“手机上不会登录账号”之间不存在必然的因果关联。

而在移动操作系统方面，Android 手机有完全可以跳过联网的选项，内地的安卓手机尚没有因不注册登录账号而不能进入启动器的现象；至于 iOS，我当然知道它相对于 Android 更封闭一些，只有 App Store 是唯一可靠的下载途径，然而，即便 iPhone 不用 Apple ID，也可以跳过登录，使用其系统的基础功能。

无论是哪家，**都没有像微软的 Windows 11 这样强制要求用户联网登录账号。**

此外，许多 Windows 用户之所以使用 Windows，其根本原因是因为自身的购买力有限，以及 PC 市场的客观因素，决定了除 Windows 外没法选；而 Microsoft 帐户及与其相关的在线服务只不过是建立在 Windows 上的附加功能。

因此，使用本地用户，不等于这样的 Windows 体验就没有“生产力”了。

微软既然有这么多年的 PC 市场实践，相信他们是不可能不知道这一点的。

其实，尽管 Windows 11 的最初几个版本并没有在 OOBE 强制联网登录 Microsoft 帐户这一限制，真正大规模引入该限制也是在 21H2 更新之末及 22H2 更新之初。

但不管怎样，强制联网登录 Microsoft 帐户这一行为，就是在故意恶心用户。

这没什么可以洗的，烂就是烂。

有答主说，微软让用户强制登录 Microsoft 帐户是“出于安全考虑”，原因是 Microsoft 帐户可以为用户管理设备的加密密钥，而普通的用户对这块不懂。

不过，从交互设计的角度来看，这点解释于本问题而言，反倒有点苍白无力。

原因是——

①登录 Microsoft 帐户并不比强密码保护的本地用户更具备安全上的优越性。**即便不登录 Microsoft 帐户，也不意味着电脑的安全性就完全不能保证了。**

对于企业电脑，这块无需多言，当然它也是一个老生常谈的话题；

对于个人电脑，典型的做法是设置高安全系数的密码，并定期更换。如果有怕密码被优盘 PE 系统破解的顾虑，可以关闭 USB 启动功能（只有需要的时候才开启），甚至可以设定只有自己知道的 BIOS 开机密码，这样，只要不更换主板，就算能启动电脑电源，没有 BIOS 开机密码也启动不了 Windows。

②在登录 Microsoft 帐户时，BitLocker 会在用户不知情的情况下自动启用（从 24H2 版本起 Windows 11 甚至会强制开启 BitLocker）。

当然这里，我并不是完全否定 BitLocker 的作用，但如此强推 BitLocker 的做法无疑有种“我是为了你好”的“爹味”，**看似为用户的安全着想，实际上并没有尊重用户的选择。**

同时，设定 BitLocker 的操作流程，对于普通用户而言确实很难称为方便，更重要的还有 Windows 上 BitLocker 数据加密的稳定性问题。

众所周知，PC 的硬件自定义程度高，厂家各异，近几年来硬件迭代升级的节奏明显加快。

而从 Windows 10 发布以来，Windows 内部的测试团队早已解散，转而外包给其他公司负责，这意味着对于消费者版的 Windows 用户而言，除非针对某一具体问题的反馈足够多，否则微软根本不会重视对于系统某一内部 bug 的解决，由此也导致，除去自家 Surface 系列的 PC 之外，微软对于大多数 PC 硬件构成的生态不具备掌控的能力。

然而，微软非要操之过急，贸然学习苹果，将 PC 与 Microsoft 帐户绑定，所以在 BitLocker 这一问题上，因电脑意外损坏维修后，即便 BitLocker 密钥正确仍无法取回数据的情况不是个例（因此，若没有额外备份，对消费者版本 Windows 的用户可谓打击不小）。

③**强制用户联网的设计，不能掩盖对于无联网用户并不友好的事实。**

有些人的 PC 不能联网是有特殊原因的，比如自己动手安装的电脑找不到合适的网络驱动，或者因为工作需要，电脑不能连接外网。

然而，微软给出的解决方案不是让他们直接创建用户进入桌面，而是在另一设备上访问如下的支持文章，以让电脑联网：

[设置Windows 11电脑时连接](https://link.zhihu.com/?target=https%3A//support.microsoft.com/zh-cn/windows/%25E8%25AE%25BE%25E7%25BD%25AEwindows-11%25E7%2594%25B5%25E8%2584%2591%25E6%2597%25B6%25E8%25BF%259E%25E6%258E%25A5-50dca26f-40d5-4c3b-853c-e972dafb7e08)

可是，对于前者，上面的支持文章给出的方法过于麻烦，小白更是手足无措，对于后者，这个支持文章的链接甚至都不一定能访问。

用户在折腾这一块花去的时间，一个跳过联网的按钮就直接节省了。

所以我实在很难理解，作为一个操作系统拥有近八成份额的科技巨头，给用户一个跳过连接网络和登录账号的按钮，真的比登天还难么？

感谢阅读～
