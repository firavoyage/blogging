## 物质

> ~~substance tan\~~~

- 物质
  - 纯净物: 由同种物质组成
    - 单质: 由同种元素组成
      - 金属单质: $\ce{Na}$ 等
      - 非金属单质: $\ce{Cl2}$ 等
      - 稀有气体: $\ce{He}$ 等
    - 化合物: 由不同元素组成
      - 有机物: 含 $\ce{C}$ 且不含 $\ce{CO2}$ $\ce{CO_3}$
      - 无机物: 有机物以外
        - 酸
        - 碱
        - 盐
  - 混合物: 由两种及以上物质组成
    - 均匀混合物
    - 非均匀混合物
