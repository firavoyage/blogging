#### 化合物

> ![][izumi] compound ~~tan~~ ~

##### 使用`树形分类法`分类

> 化合物是树干,无机物是树枝,氧化物是小树枝,酸性氧化物是叶子...
>
> ![][tree]

- 化合物

  - `有机物` 含 $\ce{C}$. 不包括 $\ce{CO}$ $\ce{CO2}$ `碳酸盐`
  - `无机物` 有机物以外的化合物
    - `酸` 能在水溶液中电离出 $\ce{H+}$. 如 $\ce{H2SO4}$
    - `碱` 能在水溶液中电离出 $\ce{OH-}$. 如 $\ce{NaOH}$
    - `盐` 能在水溶液中电离出 `金属正离子` 和 `酸根负离子`. 如 $\ce{NaCl}$
      > ![][note]
      >
      > 正离子即阳离子,负离子即阴离子.
    - `氧化物` 只含 $\ce{O}$ 和 另一种元素. 如 $\ce{H2O}$
      - `酸性氧化物` 能与碱反应生成盐和水. 如 $\ce{CO2}$
      - `碱性氧化物` 能与酸反应生成盐和水. 如 $\ce{CaO}$
      - ...
    - ...

#### 分散系

> ![][izumi] dispersion system ~~tan~~ ~

> 定义

- `分散系` 一些物质 (`分散质`) 以粒子形式分散到另一些物质 (`分散剂`) 中形成的混合物
  > 在溶液中,溶质是分散质,溶剂是分散剂,溶液是分散系.

##### 按照 `粒子直径` 分类

- 分散系
  - `溶液` 分散质粒子的直径 $< 1 nm$
    - 性质: 透明均一稳定
  - `胶体` 分散质粒子的直径 $\in [1,100] nm$
    - `液溶胶` 如 $\ce{Fe(OH)3}$ 胶体
    - `气溶胶` 如云,雾
    - `固溶胶` 如有色玻璃
  - `悬浊液` `乳浊液` 分散质粒子的直径 $> 100 nm$

> ![][note]
>
> $1 nm = 1 \times 10^{-9} m$

#### 胶体

> ![][izumi] colloid ~~tan~~ ~

> 定义 ~~(前人述备矣?)~~

- `胶体` 分散质粒子的直径 $\in [1,100] nm$ 的分散系

##### 实验: 制备胶体

- 材料: 两个小烧杯, 蒸馏水, $\ce{NaCl}$ 溶液, $\ce{FeCl3}$ 饱和溶液
- 步骤:

  1. 将蒸馏水和 $\ce{NaCl}$ 溶液加入两个小烧杯
  2. 用酒精灯加热蒸馏水, 直到沸腾
  3. 将几滴 $\ce{FeCl3}$ 饱和溶液加入蒸馏水
  4. 继续加热, 直到液体呈红褐色
  5. 制得 $\ce{FeCl3}$ 胶体

  > ![][warning]
  >
  > - 不能使用自来水代替蒸馏水,自来水中的负离子会使 $\ce{Fe(OH)3}$ 聚集, 产生沉淀.
  > - 不能过度加热, 否则会破坏胶体, 产生沉淀

- 化学方程式: $\ce{FeCl3 + 3H2O \overset{\Delta}{=} Fe(OH)3 \text{(胶体)} + 3HCl}$

  > ![][note]
  >
  > 简写:
  >
  > $\ce{FeCl3 + 3H2O \overset{\Delta}{=} Fe(OH)3 \text{(s)} + 3HCl}$
  >
  > s stands for solid "固体" ("胶体沉淀")
  >
  > or...
  >
  > $\ce{FeCl3 + 3H2O \overset{\Delta}{=} Fe(OH)3 \text{(c)} + 3HCl}$
  >
  > c stands for colloid "胶体"

  > ![][warning]
  >
  > 产物是胶体时, 不写 $\downarrow$

##### 丁达尔效应

> ![][izumi] tyndall effect ~~tan~~ ~

- `丁达尔效应` 当光束通过胶体时, 可以看到一条光亮的"通路"
  - 成因: 胶体粒子对光线的散射
  - 性质: 物理变化
  - 例子:
    - 光线透过树叶间的缝隙射入密林中
    - 电影放映机到银幕间的光柱
    - 日光从窗隙射入暗室

> 丁达尔效应是鉴别胶体和溶液的常用方法

#### 酸

> 酸与...反应

- 活泼金属 $\ce{2HCl + Zn = ZnCl2 + H2 \uparrow}$
- 碱 $\ce{HCl + NaOH = NaCl + H2O}$
- 碱性氧化物 $\ce{2HCl + CaO = CaCl2 + H2O}$
- 部分盐 $\ce{2HCl + Na2CO3 = 2NaCl + H2CO3}$

#### 碱

> 碱与...反应

- 酸 $\ce{NaOH + HCl = NaCl + H2O}$
- 酸性氧化物 $\ce{2NaOH + CO2 = Na2CO3 + H2O}$
- 部分盐 $\ce{Ba(OH)2 + Na2SO4 = BaSO4 \downarrow + 2NaOH}$

#### 盐

> 盐与...反应

- 金属 $\ce{Cu + 2AgNO3 = Cu(NO3)2 + 2Ag \downarrow}$
- 酸 $\ce{Na2CO3 + 2HCl = 2NaCl + H2CO3}$
- 碱 $\ce{Al2(SO4)3 + 6NaOH = 2Al(OH)3 \downarrow + 3Na2SO4}$
- 部分盐 $\ce{BaCl2 + Na2SO4 = BaSO4 \downarrow + 2NaCl}$

![][lucky star b]

#### 盐的形成

> 部分单质到盐的转化

- 金属单质 (如 $\ce{Na}$)
  - $\overset{\ce{O2}}{\to}$ 碱性氧化物
    - 如 $\ce{4Na + O2 = 2Na2O}$
  - $\overset{\ce{H2O}}{\to}$ 碱
    - 如 $\ce{Na2O + H2O = 2NaOH}$
  - $\overset{\text{酸或酸性氧化物}}{\to}$ 盐
    - 如 $\ce{2NaOH + H2SO4 = Na2SO4 + 2H2O}$
- 非金属单质 (如 $\ce{S}$)
  - $\overset{\ce{O2}}{\to}$ 酸性氧化物
    - 如 $\ce{S + O2 = SO2}$
  - $\overset{\ce{H2O}}{\to}$ 酸
    - 如 $\ce{SO2 + H2O = H2SO3}$
  - $\overset{\text{碱或碱性氧化物}}{\to}$ 盐
    - 如 $\ce{H2SO3 + 2NaOH = Na2SO3 + 2H2O}$

#### $\ce{NaOH}$ 的制取

- 实验室 $\ce{Na2O + H2O = 2NaOH}$
- 工业 $\ce{2NaCl + 2H2O \overset{\text{电解}}{=} 2NaOH + H2 \uparrow + Cl2 \uparrow}$
  - 原因: $\ce{Na2O}$ 成本高

#### 电解质

> 定义

- `电解质` 在水溶液中或熔融状态下能**直接**电离出离子的化合物
  - 含 `酸` `碱` `盐` `水` `活泼金属氧化物` `少数有机物`
  - 物理性质: 在水溶液中或熔融状态下, 有导电性
    > ![][warning]
    > 
    > $\ce{SO2}$ 能导电, 但不是电解质.
    > 
    > 因为 $\ce{SO2}$ 溶于水会先于水发生反应, 再才能电离出离子
    > 
    > $\ce{SO2 + H2O = H2SO3}$
    > 
    > 而 $\ce{H2SO3}$ 溶液能**直接**电离出离子
    > 
    > 所以 $\ce{H2SO3}$ 是电解质.
    > 
    > 同理, $\ce{NH3}$ 不是电解质. $\ce{NH3.H2O}$ 是电解质.
- `非电解质` 电解质以外的化合物
  > 单质和混合物既不是电解质,也不是非电解质.

  - 弱电解质
