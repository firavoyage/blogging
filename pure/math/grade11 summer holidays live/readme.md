## 1

```precious
z = 1 + bi, b != 0. z + 2/z = m. m in R. 求 m.
```

> 尝试巧算.

<details>
  <summary>[1]</summary>

  > 观察: 可化简为二次函数

  > 如果知道共轭虚根定理...

  ```precious
  z^2 - mz + 2 = 0
  z_1 = 1+bi, z_2 = 1-bi (共轭虚根定理)
    z_1 + z_2 = 2
    z_1 + z_2 = - (-m)/(1) (verta)
  m = 2
  ```
</details>
