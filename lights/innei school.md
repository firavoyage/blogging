# 最近 5 年，后悔吗

今天的话题是最近  5  年，后悔吗。原本打算聊一聊关于高考，我后悔吗。但是有感觉时间范围太小的，扩大范围之后，时间界限大概从五年前开始。

# 学业  or  学历[](https://innei.in/notes/82#0__%E5%AD%A6%E4%B8%9A-or-%E5%AD%A6%E5%8E%86)

五年前，那时我还是个高中生，正读高二。我的高中是一所非常普通的高中，这所学校原本是全市倒数的高中，但是一夜之间，他挂名了，突然变成重点高中。与其说是重点高中，但远远没有重点高中的水准，无论是从设施和教资。当时，有一种考了高分却填错志愿去了技校的感觉。学校挂名之后我们是第一届，所以学校因为各种原因，给了我们很多优惠，毕竟新校区还没建成，也不能亏待了我们。然而，等到我毕业之后，新校区也还没有建成，甚至到了下一届毕业也没有正式使用新校区。

高一高二的时候，我们的英语老师是隔壁班主任，他教学方式真的非常差，时不时开始吹起自己的牛逼经历又或者是自己儿子是清华的怎么怎么样再或者是开始膜法。与课程无关的东西，讲的津津乐道；反之亦然。所以我们班的英语可以说到了最后是没救了，英语年级排名，我们班倒数第二；隔壁班也就是他带的那个班，倒数第一。我也一直以为我也应该是没救了吧，英语常年不及格，可以说是从高中开始就没学习过英语，一直都在啃老本。然而也有很多人跟我一样。转机出现在高三那一年，换英语老师了。新来的英语老师非常年轻，教学风格与之前截然不同，慢慢的把我们的英语救回来了。她不会布置很多很多的作业，也不会很刻板的教学。总之课堂上不再是死气沉沉，偶尔还会给我们看美剧。人品很正，不像之前的英语老师一样高高在上的感觉。（之前的英语老师在某次晚自习督班时，发生过「怒斥女学生」的桥段，可见他是膜法师）到了最后一次选考考完后，只剩下语文数学英语了，那时候大概是上午语文下午数学，英语穿插其中。全班就期待的是就是英语课了，又可以看美剧了。就这样的环境下，不负众望，我们班是当时英语全年级第一，而隔壁还在垫底，每次看排名，我们总是起哄先看隔壁班的排名，又是倒数，我们都笑了。而我，也开始及格了，最后一次高考中，从第一次的  86  提升到了  110+（具体数字忘了）（浙江新高考二次英语考试机会，一般会选择在高三上和高三下）。

我不喜欢数学，不喜欢是发自内心的。不是因为数学的死板，而是难。每天有做不完的数学题，每天晚上的时间甚至可以全部花在数学上，可能还完不成数学作业，最后还是得靠抄。我开始把数学的位置放的越来越远，没时间，就抄呗。这是高三时的真实写照。一到数学课就开始犯困也是常事。当时我就很想不明白，为什么浙江的数学要考的这么难，浙江考生这么多，学校这么少，为什么还要刁难浙江考生。在这种考试的压力下，我越来越不喜欢学习。现在我终于知道，原来没学好数学是件多么后悔的事情。不是因为高考数学成绩怎么样，甚至出乎意料还考了个不错的分数。而是现在真正需要数学思维的时候，我没有。但是，如果可以回到过去，我会狠狠地抽自己吗，不会，因为环境如此。

我成绩非常的差，在班里是倒数的（虽然我们班是第一），一直在倒数前十徘徊（一共  45  人左右），七选三成绩出来后，基本可以断定能不能上大学了。而我，只有  220。从那以后，我就没有什么梦想了。希望渺茫，已经开始琢磨上职校了。以往年的分数线只有剩下三门加起来考  300  分，才有机会上二本（那也是很偏的地方了），而我平时一般语文数学在及格徘徊，第一次英语也只考了  86  分。高考前一天晚上，我一夜未眠，我想了很多，大概我的人生也就这样了吧。以至于上了考场（第一场语文），脑子一篇空白，前半小时甚至起了困意。第一场考试很失败，甚至差点没写完作文，平时都是留  1  小时写作文的，但是这次只剩下  40  分钟。下午数学，我已经不抱什么希望了。果真到了考场，做到后面大题，没有一题是会的，只能随便写写求个分数，今年题型改了，而我们平时押的也错了。把原本简单的大题和压轴题互换难度无疑是致命伤，因为我平时压轴题基本不碰第二题。而平时认为只会出简单题的题型（导数）现在放在了压轴题的位置。

出成绩的时候，我当然是不敢查的，这也注定了我的下半生。不幸中的万幸，那次成绩还可以，也超过了  300  分（指  3  门主科），按理来说是可以上二本的，但是由于我们是新高考改革第二届，有太多的未知了，上一届的分数线仅供浏览，不能参考。果然，去年的某些学校的分数线，今年涨了  70+  分。总体分数线上涨了  30+  分。幸运的是，我还有书念，也不用去职校了。

我后悔那年高考吗，不后悔，我还感觉到我很幸运。虽然现在找工作的确会遇到学历的短板，但是那年高考已经是我全部的运气了，只因为我是浙江考生。能上一个  985/211  的学校在我们这种小县城的小学校想都不敢想，众所周知，全浙江唯一一所  985/211  学校是浙江大学，而其他省对外招生招的都很少，而且浙江考生多，大部分有能力的都在杭州重点高中。

# 日常  and  自我学习[](https://innei.in/notes/82#1__%E6%97%A5%E5%B8%B8-and-%E8%87%AA%E6%88%91%E5%AD%A6%E4%B9%A0)

我是一个非常热爱计算机的人。从小就喜欢折腾系统，把玩各种软件，并且那时候开始也立志成为一个软件工程师。过了很久之后，直到上了高中之后，才真正接触编程。而在此之前，我完全没有一个大致的方向，从哪开始，又到哪里。高二那年，在家人的支持下，我买了一本  C  语言的书开始自学，这也是以后的一点基础吧。而那时候的我仍然很迷茫，不知道方向。学完了  C  的基础的之后，也正值高三了，我也不再去学一点与学业无关的东西，本身学校课程的学习就已经很累了。实在也没有精力去学其他东西了。我后悔没有早早的接触一门编程语言，以至于在短短的时间献上全部的精力也卷不过同龄人。但是，没有圈子，没有人带，对于那时候的我来说还是太难了，眼中只有一片茫然。

曾经，我幻想中的大学，是一个能和很多志同道合、厉害的人一起学习、探讨未来的地方。而现实是，是一个大部分人混日子的地方。慢慢的我开始有一个目标，至此为了这个目标，我一直贯彻到现在。一直写代码，点满  GitHub  上的方块。这个目标是因为我看了  [Phodal](https://www.phodal.com/)  的[一篇文章](https://www.phodal.com/blog/github-500-program-as-usual/)，记录了他当时学习（非科班学习计算机）的过程。而这个目标需要一个方向，而我选择了前端。虽然我没有做到每天的绿块都点亮。但是这两年来看，我没有辜负这个目标。
